从前曾是卖淫女，佛法使我获新生

Posted on 2019-08-02
末学的邪淫恶业很深重，比一般同修的更重，所以真的是万分的羞愧，万分的悔恨啊！现在我真的认识到自己的罪业有多重，真的万分感恩佛法，感恩佛菩萨，如果没有佛法没有念佛法门，我绝对会堕地狱啊！我也希望看到我这篇忏悔文的同修一定要节欲戒淫，没有学佛的人，希望你们也能早日醒悟，与佛法结缘，并且断除邪淫！末学今生就造了很多恶业，杀生，偷盗，邪淫，堕胎，妄语，恶口，忤逆不孝父母公婆，不敬师长等，真的是太多了，罪业太深重了！我忏悔一切恶业！！！愿我尽未来际永不邪淫！！！身语意长得清静！！！

末学很小的时候就手_淫包括意淫，真的年纪很小，还和一些同龄的小伙伴犯邪淫，甚至还和我亲弟弟犯邪淫，简直就是乱伦啊！末学在此泣血忏悔！到了年纪大一点，小学和初中的时候就看色_情录象和色_情书刊，那时候的意淫也越来越严重，整天沉浸在性幻想当中不能自拔，本来我的学习成绩名列前茅，到了初中就一落千丈了。考高中也没考上，就读一个私家学校，学费很高，母亲供我读书很不容易，真的对不起我的母亲！到了上高中的时候更是一发不可收拾，整天心里就想着邪淫的画面，没什么心思学习，成绩一直不好，高一期末考试竟然考了最后一名，我真是羞愧难当！

高中毕业后我就到了湖南，没有考大学，到了湖南，这里又开放又是花花世界，我的心里更是蠢蠢欲动，很快我就认识了我现在的丈夫，我们恋爱了不到一个月就同居了，而且还是我主动要和他行邪淫的！我的邪淫是多么的深重啊！我不是一个自爱的女孩啊！我后来怀孕了，因为没钱，我们就把孩子残忍的堕掉了！我向我堕掉的孩子忏悔，跪求你们的宽恕，妈妈对不起你们！我发誓今生纵遇命难，也决不堕胎了！

我的报应很快就到了，我和我丈夫到了广东，我找了一家工厂上班，因为那里的工厂都不要男工，所以我丈夫没找到工作，他就去福建了。我和我丈夫的表姐住，他表姐出去做，她认为这样赚钱快，赚钱多，我丈夫走后，她表姐就怂恿我让我也去和她一起做不正当的行业，我当时年纪轻很害怕，开始很犹豫不想做，但是我敌不过业力，敌不过贪婪，敌不过报应。

我丈夫这时突然说没钱回广东了，在福建没找到工作，让我给他寄路费，我当时根本没钱，我思来想去就决定走邪路赚钱了，就这样我迈上了令我耻辱的邪路！刚开始内心很挣扎，很害怕，很不情愿，可是过了没多久我就被金钱和欲望吞没了，我看到身边的人买名牌，吃喝玩乐很自在，我也想和他们一样，我的丈夫回来后知道了我去做不正当职业赚钱，开始很反对，可是后来他看见我拿回来的钞票，不再反对，还鼓励我到大酒店去多赚点钱，我就这样在邪淫的路上，一边数着钞票，一边糟蹋着自己，当时我竟然毫无廉耻，我竟然觉得无所谓，我知道这是我的报应，从小邪淫，婚前同居，还有堕胎，所以我才会任人玩弄，才会备受凌辱！我今天向佛菩萨泣血忏悔啊！我以后就算死，也不再做邪淫的事了，并奉劝天下所有女子，自尊自爱，不要被金钱和贪婪驱使！

后来我又堕了两个小孩，不是和我丈夫的，他们被我残忍地杀害，就在我堕第二个孩子的时候，因为吃药没堕下来大出血，我上班的时候，身下出血把沙发都染红了，到了医院，严重失血，医生马上给我输葡萄糖，输了两个小时，终于好转，然后又一次把我体内剩余的婴儿身体掏出来，我是何等的残忍，这样对待我的孩子，我真的不配做母亲，孩子啊，妈妈向你们真诚的忏悔！！！这一次堕胎差一点要了我的命，我现在知道肯定是我堕的孩子非常的怨恨我，想要我的命啊！奉劝天下所有女性千万别再堕胎了！

从堕胎以后我的身体日渐衰退，更糟糕的是我患上了慢性盆腔炎，我知道这是我堕胎和邪淫的果报，我的这种慢性盆腔炎根本没办法根治，吃药打针只是暂时控制，过没多久又犯，反反复复折磨我，每次我犯病的时候，肚子痛的要命，那种痛就象堕胎时被清宫的感觉，每次痛的我浑身冒冷汗，更有甚者，有几次痛的我满床打滚，真的是自作自受啊！诸位同修们，这就是我的现世报啊，大家千万别步我的后尘啊！

后来我又一次怀孕，我骗丈夫是他的孩子，其实是我和别人的，因为我怀孕了，而且身体很不好，各种炎症，精神也萎靡不振，没办法，我的丈夫和我一同回家了，我第三次堕胎了，这一次因为坐火车受凉，我又出血不止，回家又是打针又是吃药，过了很久才好，当中我所承受的折磨和痛苦，只有我自己能体会，真的很痛苦！

回家以后，不是这里病痛就是那里不顺，总之种种难题种种烦恼，活着没有目标没有方向！婚姻也不顺，和丈夫好不容易才结婚，婚后我和婆婆关系一直不好，她看我不顺眼，我看她不顺眼，我怀孕期间和婆婆争吵，顶撞她，也经常对丈夫发脾气，我的丈夫以前脾气很好，可是后来他脾气变的很暴躁，甚至有一次还动手打我，其实回想起来，这不是没有原因的啊！我深深忏悔，并代我的丈夫忏悔，忏悔我们婚前同居及堕胎的罪业，并忏悔我们非时非地行淫的罪业，忏悔我们几次在我娘家邪淫的罪业，忏悔种种邪淫，忏悔种种恶业！！！我发誓远离邪淫，努力改过，好好念佛！

让我最痛苦的是我的病一直反复发作，我走邪路的几年中，我是赚了钱，可是我的钱不是治病就是赌博输掉了，根本没存钱，我为了金钱为了欲望，欺骗别人，做风尘女子，用身体当赚钱的工具，我得到了什么，我得到的只有精神和肉体的折磨和无边的黑暗痛苦啊！邪淫只有损福折寿，只有凄惨的果报等着犯者去承受！

万幸的是我往昔有点善根，我想我纵然有点福报，也早被邪淫耗尽了吧！我30岁终于遇到了佛法，遇到了明灯，我黑暗的日子终于结束了，我现在很想哭，我不知道用什么言语来表达我对本师对三宝对阿弥陀佛的感恩之情！我2010年在网上看了很多关于佛法的内容，我一下子就被吸引了，我就开始念佛，过了没多久就开始吃素，我吃素前闻到荤腥就想吐，看见也不舒服，我的吃素因缘很快就到了，佛菩萨很快就加持我了！很感恩佛菩萨！

刚开始学佛我读了很多经。念了很多咒，和许多初学同修一样走了很多弯路，后来我有幸闻到净空老法师讲经和开示，我看了很欢喜，慢慢放下其他法门，专修净土法门，念四字圣号，现在过了两年多了，我的病竟然全好了，我没有打针也没吃药，我的盆腔炎竟然没发作了！........

我丈夫开始和我吵架，还撕毁我的经文，说一些毁谤三宝的话，不让我学佛，甚至说我如果再学佛，他就和我离婚，我开始还和他争辩，后来看了老法师的开示学佛要随顺众生，不要和他们争辩，要用行动去感化他们，我就不和他争了，我仍然念佛诵经，有时去放生，还每天为鬼道众生施食，本来我很胆小怕黑，不敢施食，可是后来看很多师兄施食施的很好很欢喜，我就决定施食了，鬼道众生其实一点都不可怕，他们很苦，我们学佛要学佛菩萨的慈悲心，我施食一年多了，不仅没什么不好，身体也好了，气色也好了，相貌也变好了，越施食越欢喜，学佛学的很欢喜！我丈夫不再那么反对我了，只是不当着他的面念佛诵经就行了，他也不会多说什么，我知道引起我丈夫毁谤三宝，我有很大责任，所以我深深忏悔，我也代我的丈夫向佛菩萨忏悔！以后弟子一定学会随顺众生！也愿我的丈夫得三宝加持，早日学佛念佛求生净土！

我为我堕胎的婴灵立了牌位，并念佛给他们回向，还抄写念诵佛经为他们回向，有一次我做梦，梦到有一个小男孩，进到我房间，我就坐起来跟他走到外面，他站在院子当中对我挥手，我就念阿弥陀佛，他也跟着念，一边念一边对我挥手，我感觉他在和我道别，我就对他也挥手，突然我看见观世音菩萨从西方缓缓而来，一瞬间就到了我面前，有两层楼那么高，穿的白衣服，
天上有一朵粉红色的莲花，那个小男孩一下子飞到空中，上了那朵莲花，他的体貌也变了，开始是个普通小男孩，到了莲花上面，就变成古代画中童子模样了，他对观世音菩萨礼拜，然后就走了，我醒了感到很欣慰！不知道他是不是我堕掉的孩子！

至今我学佛两年多了，我的生活身体事业都有很多变化，我确实也认识到了自己的罪业深重，我也经常忏悔！以前我经常做恶梦，还经常做邪淫的梦，肯定是邪淫感召的，我虽然身体没去邪淫，但淫心未断，所以经常梦见淫魔，邪淫事，我就经常发表断除邪淫的帖子，并且修不净观，现在少很多了，再加上一心念佛，求佛加持，我相信我一定可以战胜邪淫，断除淫心！

我以前虽然念佛但信愿不坚定，求往生的心也不恳切，后来我通过多听净空法师，定弘法师还有刘素云老师的讲法开示，终于转迷为悟，我们每个众生本来就是佛，为什么还不念佛去做佛，为什么还把这个虚幻的六道轮回当成真的呢？我也终于体会到了弥陀的本愿功德，弥陀的大慈大悲，对我们这些恶业深重的众生都是一个也不舍弃啊！我们不要再犹豫，只要把手伸给阿弥陀佛，让弥陀牵着我们回家吧！

通过我对极乐世界对阿弥陀佛的信愿坚固，发愿求生净土的心真诚恳切，愿意随时跟佛走，愿意放下一切，努力的念佛，我真的感应到了弥陀的加持，真的体会到了和佛感应道交，一次我拜佛时，抬头就看见阿弥陀佛像的头部出来一个万字符，金色的，后来接着又出现一个，我真的很感动！还有我家里每天为众生播经，我和我丈夫租的房子，我们住的房间外面窗台放了一个播经机，楼上放了一个念佛机，放的老法师念佛的佛号，有一天傍晚我上楼有事，当我上到三楼猛一抬头，我惊呆了，我竟然看见一朵车轮大的粉红色莲花，我开始以为眼花了，我又仔细看看，莲花还在那里，我呆呆的望着莲花有几分钟，从那开始我更加坚定了往生的信愿，也非常的欢喜，每天念佛念的很快乐！

我的梦境也有很大改变，初学佛时老做恶梦，很害怕，有好多次梦见恐怖的梦，还梦见很多手抓我，还看见一个白面人来掐我，还有很多次被什么东西压的喘不过气来，还看见骷髅头，女鬼什么的，我在梦里念佛，念观世音菩萨圣号，我一念就没事了，有时就醒了，有时还看见佛菩萨真的现身了，我真的太感恩佛菩萨了，象我这样一个造作五逆罪的人都能得到佛菩萨的加持，都能有机会出离六道轮回，我们还有什么理由不好好念佛呢？能不报佛恩吗？慢慢的，我不再做恶梦了，有时梦见莲花，有时梦见佛菩萨，有时梦见念佛，有时梦见飞，那种飞的感觉实在太美妙了，当然同修们不要执著梦境和感应，梦见也不要老想着，也不要欢喜激动，老法师说我们就是看见阿弥陀佛也不要激动，激动就错了，因为阿弥陀佛本来就是我们的自性嘛！我还梦见净空老法师了，老法师对我笑的很亲切，很慈祥，我感恩老法师给我指引了正确的光明大道.......。

以前我梦见众生都对我很凶，很可怜，通过我的施食念佛，还有给他们播经放佛号给他们听，通过我诚心的忏悔，并对他们做简单开示，劝他们求生净土，而且我经常给他们立牌位，现在我梦见众生，他们都挺高兴的，有的还对我双手合十，有的在念佛，总之没那么忧愁痛苦了，他们一定也明白一些佛法了，也念佛了，真为他们感到高兴啊！当然我做的还远远不够，我还要继续努力，我的业障还很深，修行路上的考验还很多，不过我想我怀着一颗坚定的心，无论如何，什么都不能阻挡我往生净土的决心！还有我以前念佛妄想很多，后来我按照十念法念佛，效果好多了，念十句计数，然后再念十句，反复这样念，心里计数就行，印光大师的十念法太好了，值得我们大家学习，尤其念佛妄想多，不得力的同修们，我用十念法念佛，念的很欢喜，而且身体感觉很轻快，很舒服！

我今天写这篇文章，一方面诚心忏悔我的罪业，一方面想供养大家，让已学佛的同修坚定信愿，不是我们念佛不灵，不是佛菩萨不加持我们，而是我们的心不恳切，不至诚，求生净土的愿心不够，信愿不真不诚，我们要发广大菩提心才能与佛心相应啊！学佛不能为自己学啊！

我建议大家每日念莲池大师的西方发愿文，真诚的发愿，发大菩提心！“弟子愿从今日起，远离恶法，誓不更做，勤修圣道，誓不退堕，誓成正觉，誓度众生！”这是西方发愿文上的话，我们就要发如是菩提心！并且要放下一切，愿意和佛走，佛什么时候来接，就什么时候走，一定要有这样的心态才行，劝同修们不要放不下了，常常观身不净，观受是苦，观心无常，观法无我，还有想想地狱的苦，饿鬼的苦，做畜生被宰杀的苦，互相厮杀啖食的苦，还有生老病死苦，世间真的有乐吗？没有啊！回归我们的自性才是真乐，涅槃才是真乐，到极乐世界才真自在快乐啊！没学佛的同修愿他们引以为戒，断除邪淫，也能念佛求生净土！弟子至诚忏悔，感恩佛法救了我！！！

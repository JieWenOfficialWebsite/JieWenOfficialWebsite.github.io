念诵《楞严咒》戒邪淫最快，非常有效【转】

　　阿弥陀佛，在爱情里婚姻生活美满，这是最幸福的状态，但是，有一个原则必须不能触碰，一旦触碰，万劫不复!


　　有人问菩提：在爱情里最重要的是什么?夫妻之间需要谨守什么原则，才能让爱情圆圆满满。


　　菩提想告诫各位同修们：无论是信任，还是相亲、相爱，这都是夫妻之间理应遵守的最基本原则，但是，还有一个非常重要的原则，那就是一定不能犯邪淫。


　　一旦夫妻中任何一方犯了邪淫，和谐美满的婚姻就会产生裂痕，继而严重地损害了婚姻中的福报!后果不堪设想!


　　1邪淫的果报——婚姻不和，感情不顺


　　邪淫最损福报，很难成就世间利益，这好比数十年辛苦的基业，一把大火就可毁于一旦。学佛人必须要谨慎。


　　那到底什么是邪淫?简单而言，就是夫妇之外的男女关系，一切不受国家法律或者社会道德所承认的男女关系，均为邪淫。


　　经常犯邪淫的人，往往会有以下几种果报。


　　1、学业功名受损。只要淫心一起，福报就如同飞流直下的瀑布一样，片刻消耗殆尽。正在求学之人，精力不如从前，成绩也会一落千丈。官场之人，哪怕在提携的前一刻，也可以黄粱一梦。什么平步青云，飞黄腾达，恐怕也是与你无缘了。


　　2、身体变得体弱多病。人人都知道，犯邪淫最伤害身体，欲念缠绕，连绵不绝，内心如火般烧，身心饱受折磨，每天疲惫不堪，念佛也打不起精神。


　　3、最常见的是家庭婚姻不和。生活中各种各样的例子，如果有一方出现外遇，结果就会引发家庭婚姻的不和，严重还会引发家庭危机。


　　世间名利、事业、健康、钱财、福报和淫-欲本来就水火不相容，千万不要因为邪淫，而影响自己诸事不顺，丢了钱财，损了福报，得不偿失啊!


　　2持诵楞严咒，是对治淫-欲的根本大咒!


　　学佛什么最难，感情事最难断!修行人第一难戒的就是淫-欲，因为意念一直都在，非常难驱除!


　　如果你直到现在还戒除不了邪淫，请试试这个方法：持诵楞严咒，是对治淫-欲的根本大咒。


　　楞严咒太长，一般学佛弟子不容易记住。如果觉得太难，也可以只背诵楞严咒心咒，很短，效力不可思议!


　　楞严咒心咒： 都鲁雍瓦达,大地亚他。嗡，阿努利，毗舍提，皮啦，跋阇啰，陀利，磐陀磐陀尼，跋阇啰，谤尼泮，虎星，都卢雍泮，司瓦哈。


　　念诵的次数，可以根据自己的修行时间进行安排，每天持咒1遍、3遍、5遍、7遍、14遍、21遍，都是可以的。


　　持诵楞严咒，是对治邪淫的因缘，坚持下去，欲念清净了，身心也轻松了!


　　除此之外，修行上要持续念佛、诵经，亲近善知识，生活中最好断绝葱蒜韭菜、酒肉，坚持吃素，培养一颗清净心，体能清净充实了，自然就远离邪淫了。


　　愿天下一切众生远离邪淫，远离邪法，从而亲近善知识!南无阿弥陀佛!

邪淫 手_淫 婚外情 忤逆-让我历尽人生坎坷曲折【转】

Posted on 2019-08-02
　　写这篇忏悔文时，我仍然心存犹豫，尽管很早就在佛前发愿要这么干，却依如从前那样迟疑着要不要现在就写，要不要就这么无情地撕开心底的那个伤疤、毒瘤。确实，过往多年的邪淫史不堪回首，甚至连我自己也不肯直面。
　　
　　要不是格格师兄一直说如果不把邪淫事点点滴滴的写出来，你的忏悔都没忏到位，我才下定决心写，决定不要这张脸，写出来。
　　
　　在这里，向诸佛菩萨真诚忏悔自己的散心懈怠，无比感恩耐心阅读此文的诸位菩萨、仁者为我慈悲见证这份迟到的忏悔！祈求佛力加被，愿一切造作邪淫罪业的众生所应承受的成熟和未成熟的业果都由我弟子慧兴承受，愿以此文及我三世一切之善根回向他们一念回头觉悟，发趋一乘，念佛永脱苦海！愿有缘见闻此文者都能断恶修善，远离一切邪淫及恶行恶念，起大精进如有犯者，则自悔过，后不再造；如无犯者，当恭慎保持端心正意，洁身自爱！
　　
　　我以邪淫的惨痛教训深刻感受到“万恶淫为首”这句话的无比分量。邪淫如鸦片、毒品般具有极强的“瘾”，一旦心存侥幸染上，就欲罢不能，愈陷愈深，扭曲人性，直至彻底断送一个人所有的前程包括生命，其害无穷，数不胜数！邪淫者必折福折寿，现世受种种苦厄花报，命终定入地狱百千万劫无从出离，不可不戒！
　　
　　我是来自广西一个偏远的小山村， 70年代的人，那个时代的人都重男轻女，我是家里的长女，，从小脾气就倔强，喜欢跟父母家人顶嘴，家人让我往东，偏往西，因此没少挨打，因为这个原因我特别的憎恨父母亲，他们越打我，就越恨，还喜欢说刺激的话，让父母家人伤心难过掉眼泪，曾经有一次起了自杀的念头，想一死了之，再也不用呆在这个没有温暖家庭。那个时候的自己完全被憎恨心蒙蔽了，看一切都不对，只有自己是对的，是个极端忤逆的人。在此长跪向父母亲忏悔，对不起爸妈，我不应该怨恨你们，咒你们，一切都是我错了，我错了，在此发誓在您们的有生之年好好孝顺，禰补以前的过错，请您们原谅这个忤逆的女儿！ 忤逆父母罪就是犯天条，会被削减福德，一生事事都会不顺!
　　
　　染上邪淫这个恶习肇端是在我的童年记事时期， 那个年代条件不好，家里穷，一次穿了条过小的裤子，勒得下身很紧，走路摩擦起了快感，这种感觉让我久久回味，一发不收拾。有时候为了找这种感觉，特意到没人的地方拉紧裤子找这种感觉。自此邪淫就在心中扎下了根。
　　
　　上大学时认识了前夫，他是外语系的，比我大7岁，谈了2年，在这2年的时间里跟他发生了性关系。经常去学校外录像厅，包厢通销看黄碟，在学校的宿舍里，教室里，树林里，河堤上做邪淫的事，让邪淫的气味污浊了虚空法界，污浊了在那里修行的众生。毕业时因为论文过不了关，差点拿不到毕业证。毕业后与前夫分手，在市里找了家保险公司上班，与暗恋自己的高中男同学在一起同居，后来经双方父母同意定婚，把关系定了下来。同居期间前夫不断地打电话来要求复和，经不起他的甜言密语，也是自己淫心重，朝三慕四，偷偷地从未婚夫家留走，跑去东莞。没有告诉未婚夫去那里，娘家人也没有说，就这样偷跑了。未婚夫为了找到我，找遍了所有认识我的人及关系网，还跟自己的父母亲闹翻了，从此不再喊爸妈一声，他一直认为是他爸妈对我不好，我才走的。其实是我跟别人跑了。我欺骗玩弄别人的感情，是天底下最无耻最不要脸的，无德无行的恶女人。
　　
　　由于前夫 精通外语，业务能力强，在公司的地位直线上升，钱也跟着往上涨。跟前夫结婚时间不长，他就变心了，在我生孩子期间与公司同事常去KTV,夜总会玩，找做台。他手下的业务小姑娘，好几个跟他不清不楚。后来有个小姑娘耐不住了，在04年年初三以送礼为借口来看他，我才知道他们这种不正当的关系，当时把我气疯了，把这件事闹到了老板那里，把那个女的开除了。因为前夫的不悔改，一天晚上等孩子睡着了，手里拿着水果刀想到割脉自杀，被人发现后，在外人的一再劝阻，安慰下，放弃了自杀。此后感觉这段婚姻太让人伤心难过痛苦不堪，于是离婚了。离婚后前夫跟他手下另一个小姑娘结婚了。-------这就是我欺骗别人的感情，抛弃别人的现世报，这个报应来得真快，让我尝到了那种生不如死被背叛的滋味。
　　
　　在与前夫婚姻期间，他一直对我很冷淡，高中男同学后来得知我的情况后，为我一直报屈安慰我。一来二去的，我们又在一起发生了几次关系，后来我认识到自己这样做是不对的，跟他断了联系不再来往。
　　
　　离婚后，我一个人过，经常想东想西，很快就得了忧郁症，精神常常愰愰惚惚的，直到现在忧郁症还没有好完。想到前夫这样对我，起了报复心，于是去勾引他最好的朋友，以为这样就可以报复到他了。其实还是自己错了，冥冥中又加重了邪淫的罪业。还染上了SY，意淫的坏习惯。
　　
　　这个时候遇上了佛法，于是把所有的精力都花在学佛上，刚学佛不久认识了一位佛友，他把佛法讲得头头是道，是位老修单身，认识了2年，在一起很谈得来，他说想跟我结婚在一起学佛修行，可以组织一个佛化家庭，这样对学佛帮助大。我信以为真，结果他骗了我并不想结婚，就是想俩个人在一起住，有个伴给他做饭吃。因为学佛了知道在一起住没有领证是属于邪淫罪，离开才是正确的选择。如今证实了当初的选择是对的。他后来又用同样的方法骗了另一个女的，差点被那个女的男人揍。
　　
　　-----感情又被别人伤害了1次，这就是邪淫的果报，自己也是罪有应得。
　　
　　1. 邪淫恶报--------福禄几被削殆尽，历尽坎坷曲折
　　
　　毕业后一直找不到合适的机关单位，被迫去广东打工。在那年代农村出个大学生不容易，还找不到工作，在老家真的让父母亲难堪，丢脸。至今为止，工作单位换了10家之多，涉及行业不一，工作内容不一，无直接相关性，职业定位与职业生涯规划几乎没有，一片混乱状态，找不到适合自己长期稳定发展的工作，最长的1次不超过3年，总是在工作出成绩时，就会出现不想做的念头，任谁来劝都不想做下去。想要加工资总是遥遥无期，看不到希望，比自己能力差的人可以拿到高工资，跟老板谈加工资都是被拒绝的。--------希望看到此文的大学生及青年学子们，你们一定要引以为戒，不要谈恋爱，更不要发生性关系，大学期间是用来好好学习知识，将来出社会报答天地父母恩的，不要浪费了光阴金钱，消耗自己的福报，毁了美好的人生前程。
　　
　　2. 邪淫恶报-------- 多种疾病缠身，身心极度虚弱，近乎神经质，几近自杀
　　
　　A, 白天多幻觉，夜梦沉沉，多淫思邪梦；
　　
　　B. 心跳加快，胸闷憋气。
　　
　　C. 记忆力极差，健忘成性，反应迟钝，理解接受力低；
　　
　　D. 自信心不足，做事乏力，缺少斗志激情，经常虎头蛇尾或轻易半途而废，无有耐性；
　　
　　E.一度整天精神恍惚，身体沉重，晚上梦厓。
　　
　　F. 头——发质枯黄，暗弱无光泽，频繁脱发掉发，多白发。
　　
　　G. 下身时常潮湿不净，有异味。
　　
　　H. 经常性浑身软弱乏力，极易疲劳，身体沉重。
　　
　　I. 眼——视力直线下降.现在是550度
　　
　　J. 耳——长期耳鸣,头昏，有时自己晕过去都不知道，醒来时还想自己怎么躺地上了。
　　
　　K.说话中气不足，声音小，说话上不来气，像被人卡着脖子发不出音。
　　
　　L. 毁容----脸上长满了痤疮，让看到的人产生厌恶感。
　　
　　M.（堕胎）-------得妇科病。跟前夫怀第一胎时，由于刚结婚经济条件不太好，怕有了孩子拖累，2人一致同意堕掉孩子。生下老大后又怀上了，也不敢要又堕了。2次打胎，让身体夸了，原来110斤变成了90斤，落下很多妇科病，每天腰酸背痛腿抽筋，稍微干些活就会让腰痛得都直不起来，比一个60岁的老太婆身体还差。
　　
　　3. 邪淫恶报--------淫心重感招鬼魅上身发生关系
　　
　　由于淫心生而感招了邪魅上身，在梦中常常与之发生关系，第二天醒来身体疲惫不堪。好多次在梦中都不愿意做这件事，硬是被逼迫发生关系。这件事发生了2年之久，到后来晚上都不敢上床，不敢合眼，害怕一睡过去，那个鬼魅就来了。最惊现的一次，夜里一直不敢睡觉，实在是熬不住了，刚一合眼就来了，上来就卡住我的脖子说让你不睡觉，被施法动不了身说不出话，情急之下想起佛号，默念南无阿弥陀佛，到第二声时忽然能动了，鬼魅也不见，被吓得一个晚上不敢睡，一直念佛号到鸡叫时才放心睡。于是每天晚上念佛号来跟鬼魅做争斗。至今邪魅仍未离去。-------验证了佛法里说的一切唯心造，什么样的心性感招什么样的众生亲近，佛法是真实不虚的，佛号的威力也是真实不虚的。
　　
　　4. 邪淫恶报--------做生意失败亏本
　　
　　离婚后学别人做生意，一样的生意别人搞得红红火火的，轮到自己不管怎么努力，一点起色都没有，亏了好几万。
　　
　　4. 邪淫恶报--------懒散懈怠，注意力不集中，心极散乱，极不清净，昏沉掉举嗜睡，学佛难精进.
　　
　　有幸闻到圣贤教诲和佛法后，虽能生欢喜心接受，然总归难以落实。不听经闻法，不诵经念佛拜佛还好，一发心去做这些，就开始犯困打瞌睡了，即便勉强为之也总是心不清净，注意力难以集中，心还在别处胡思乱想。纵能一时发心去熏修佛法，却总是生不起长远的精进心，生不起真正的信心，总是三天打鱼两天晒网。每天早晚作息不能规律，早起精进非常困难，日常修行仍然不免散乱懈怠，
　　
　　限于篇幅所限，尚不能逐一详细展开。凡上述我所列举和提及到的种种邪淫过恶，以及其他一切我有意无意中遗漏的、遗忘记不起的种种大小邪淫过恶，乃至我无始劫来宿世所作一切大小邪淫过恶悉皆忏悔认罪！
　　
　　愿三宝加持，愿我弟子慧兴从今以后，乃至尽未来际，生生世世后不再造，坚决、彻底、干净地远离一切邪淫的心行；
　　
　　愿常存清净愿，发菩提心，为度众生此生往生西方，成就无上菩提，如佛度一切；
　　
　　愿一切造作邪淫罪恶的众生都能因我弟子慧兴真诚心断恶修善，而净除宿障，善根增上，起发无上菩提心，念佛求生净土！
　　
　　南无阿弥陀佛！

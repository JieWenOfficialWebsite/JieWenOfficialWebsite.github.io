陈冠希的爷爷叹“搞了几十年教育，却无法教育好子孙”

  • 2009年03月26日15:19 中国企业新闻网 作者：郑播德

　　陈冠希的爷爷，干了一辈子教育却剩下悲叹

　　陈冠希的爷爷，名叫陈邦平，曾在香港政-府教育署工作，作为资深公务员，也有不少积蓄。
　　陈爷爷虽有5名子女，孙儿众多，他却当陈冠希为“掌上明珠”。

　　陈冠希早年性格反叛，陈爷爷却一直称赞他本性善良，还曾批评其他孙子不脚踏实地。陈爷爷每月都给孙辈们几千元作零用钱。

　　但是，有位孙女竟然在没零用钱时偷了同学钱财，让老人家非常痛心，大叹“搞了几十年教育，却无法教育好自己的子孙”。

　　遗嘱遗产风波弄到报警

　　陈爷爷本来打算将最值钱的复式单位（复式套房）留给陈冠希，并私下立好遗嘱。

　　但陈冠希的父亲陈泽民却认为：自己的妹妹的儿子更需要房子，便拒绝了老父好意。
　　此后，却引起了争产风波。

　　有一位亲戚指控说，陈爷爷在住院期间，遗嘱却无故消失，怀疑有人意图要让房子从分给一个孙子变成平均分配。

　　还有人报警，申请“搜查令”，要求搜查每个拥有陈爷爷家钥匙的人。

　　而陈冠希的姑姑（陈泽民的妹妹）每次探病后，都拿一张写满字的纸走出病房，并对其儿子说：“不要乱放，这可是证据。”

　　陈泽民也私下教妹妹快点找律师搞定，开个联名户口，商量钱怎么分。更教妹妹不要钱，要房子划算。

　　陈冠希的父亲陈泽民屡传绯闻，又面临破产

　　有记者曾向香港同行打探陈泽民的发家史。

　　上世纪90年代初，陈泽民在澳门当财务顾问，随后炒楼发家。为突显其富人身份，陈家曾住豪宅和别墅，但后来证实都是租的。有传陈泽民更坐拥两部劳斯莱斯等6部名车。

　　2008年，中银香港向陈泽民所在公司追讨欠款，追款时称，2000年，陈泽民公司向中银香港借款2350万，用来偿还公司的透支及按揭贷款，10年半内还清.但到去年5月，还拖欠1900万，原告去年6月曾发律师信给被告要求还款，但被告仍没归还。

　　陈泽民本人日前证实，“金融海啸”导致银行收紧信贷，令其公司资金周转出现问题，他将正式向法庭申请破产。

　　在过去多年中，陈泽民更是被媒体反复传言“同性绯闻”，与多位男子关系复杂，阴影难除。

　　陈冠希身陷“艳照门”遭遇“两颗子弹”威胁生命

　　相信陈邦平先生是一个正直、作风严谨、干净的人。但由于教育的疏忽（当然也离不开社-会的影响、子女自身的把握等问题），到了陈冠希身上，男女关系方面，竟然“滑”到了“随便之极”的地步。后来爆出与多位女人的“艳照门”事件，也就是“纸包不住火”的必然结果了。

　　前不久，陈冠希在宣布复出后，竟连续两次接到“死亡子弹”恐吓信，更成为海内外媒体追踪对象。

　　这一幕幕、一环环，恰似电视连续剧，一波未平一波又起，让陈家人未曾“消停”过，媒体上尽是负面报道。

　　如此状况，岂不令人唏嘘：

　　作为教育工作者的陈邦平先生，见到子孙如此状况，如何心甘、平复？

　　培德培福、兴家兴业的根本良方——戒“邪淫”

　　作为读者的我们、为人父母的我们、为人子女的我们，又该从这一幕幕“闹剧”中吸取什么教训？

　　当今时代，是个“泛娱乐化”时代，人人都会从电视、互联网、图书、现实生活中，见到一些不雅的、伤风败俗的事情——尤其以男女关系、男男关系为最，在没有“免疫力”、没有坚固“定力”的情况下，就会由“羡慕”而走向“模仿”、实践，导致一系列不可收拾的可怕结果。

　　在佛法中，这类不正当的男女关系（非夫妻关系）、男男关系、女女关系、自慰、嫖妓、包二奶等，都属于“邪淫”的范畴，是一定要“力戒”的重大“恶业”之一。

　　不论是谁，只要一沾上“邪淫”，就会导致种种“昏迷颠倒”的事情发生，对学习、工作、经商、从政等产生巨大的恶劣影响：

　　常常会导致“决策错误”、判断失误、头脑晕乎、身心亢奋难宁等“中毒中邪”现象，导致生意失败、被骗钱财、糊涂丢官、贪污腐败、家中人种种不顺、生病（包括性病、艾滋病等）、牢狱之灾、身败名裂，甚至车祸、“暴死”等等恶劣情况发生。

　　“邪淫”是剧毒——毒己、毒人的“砒-霜”；

　　“邪淫”是无底的黑洞——破财、销金的“捷径”；

　　“邪淫”是可怕的“刮骨钢刀”——破身、摧毁健康的“利器”；

　　“邪淫”是“速败家的无间道”……

　　所以，孔子曾说:“君子……少之时，血气未定，戒之在色。”

　　我们每个人都应该发愿做个心行自律、洁身自爱的正人君子，才是“保身”、“保财”、“保命”、培福培德、兴家兴业的根本良方。

　　过去由于无知、贪色，已经犯下错误的，应急速刹车、悬崖勒马，改弦易辙，改了就好，庶几有救！ （郑播德）

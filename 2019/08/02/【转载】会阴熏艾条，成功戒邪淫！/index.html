推荐：会阴熏艾条，成功戒邪淫！

云中凤

这几天，总有网友询问笔者，自己的邪淫自慰欲总是控制不住应该怎么办？正好此前广东网友向笔者推荐了一种方法，治疗邪淫简单有效，神奇无比，就是用艾条熏会阴穴，就可有效根治邪淫、清净事者淫-欲心，熏一次即可见大效。笔者认为这个方法很有道理。

为何呢？因为首先，熏艾绒具有驱邪避凶的作用，二者，通过熏艾绒的烟渗透海底轮的会阴部，可有效驱逐附着在那里的阴性物质。很多人之所以淫-欲强烈，主要是邪淫心感招淫魔附在阴部吸精造成。戒邪淫对年轻人来说确实极不容易，可是修行人不戒邪淫是不能成就的，更不会有好运气的。

推荐此法的网友妻子，就是用这个方法有效戒除了淫-欲，并且已经有网友经笔者介绍使用此方法后，收到了很好的戒淫效果。但须知，戒邪淫还是要在心上下功夫，不能一边熏艾条，一边想淫事，如此下去只会自欺欺人、自毁慧命。那怎么样来熏艾条呢？请听笔者告知：

买一个坐便座，下面放一个纸盒什么的，只要能将艾条盘托起来，距离你的会阴部5寸高即可。熏烟不要过烫，也不要过凉，否则会影响效果，你来试一试，看看效果怎么样？相信你会惊喜的。如能同时做真诚忏悔，戒淫的效果更会加倍和加速的呵呵！

愿以此功德，回向邪淫众生，愿你们都能戒邪淫，行正道，做好人，成菩提！南无极乐世界阿弥陀佛！

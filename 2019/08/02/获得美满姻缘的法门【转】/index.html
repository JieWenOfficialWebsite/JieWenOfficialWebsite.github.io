戒文~获得美满姻缘的法门【转】

Posted on 2019-08-02
当代年青人最大的烦恼就是如何找一个如意伴侣。在元亨网呆了一段时间，在算命的网，自然能看到更多为情所苦的人。

作为一名佛教徒，很乐意将一些知见与法门与诸位有缘善信广而分享。
上元佳节，是中国传统的情人节，在这样一个特殊的日子里，希望送给大家一份美满的姻缘。

很多人眼里，与佛结缘，就是四大皆空，必定谈情囧囧色变，以为接触佛法之后便应立志超凡脱俗，不可再有世间恩爱，误会不浅。
求出世自然是学佛最高的祈愿，但若无法放弃世间种种，修习佛法仍是最好的处世方针，若能确实遵一二条教义，自能让世间福报源源不绝，让自己与亲眷受益无穷。

对佛之信仰，是接受悉达多太子所表述的世界观，从而形成的一套人生观与价值观，于是成了佛弟子；这与接受共产主义思想而形成一套的世界观、方法论如是成为共产主义者一样，不过是认定一种理想，发心为之奋斗。学佛之言行，便是我们作为圣言追随者对幸福生活孜孜以求愿望的表征。

什么是真正的爱情和婚姻？
1.贪色或贪财，是不会幸福的，不但不会幸福，即便宿福深厚，不幸也会接踵而至。
2.断除邪淫是幸福的根本。邪淫的危害很严重，它会大大减少幸福爱情和美满婚姻出现的机缘，使有可能白头到老的夫妻变成只是一段缘分。
3. 真正可靠的爱情和婚姻是建立在2个人有共同的信仰和脚踏实地的生活之上的。共同孝养父母，奉事师长，相互之间共同关心和爱护。
4．美满的婚姻是夫妻双方的事情，不光配偶要有一定的人格修养、道德素质，我们自身的品质如何也很重要，所以，要想拥有幸福的生活，时刻不能忘记对自己个人情操与修养的培养。

美满姻缘，是我们自己争取的。不管何种方式，前世还是今生。佛教故事都听得很多，什么前世的500次回眸……前世葬你的人……

姻缘有缺，势必德行所欠，甚或正是邪淫所感（邪淫，包括一切除夫妻之外的不正当的男女关系，之中包括婚前同居、婚外恋、卖囧囧囧淫、嫖囧囧囧娼，谈论不净之事，看不净之书和音像制品）。

这里我介绍给大家求姻缘的确实有效的法门，有心者不妨一试。

确定自己的姻缘标准，写了一篇“求夫（求妻）疏 ”向菩萨求姻缘。




求夫（求妻）疏



南无大慈大悲观世音菩萨，由于弟子累生累世邪淫，不孝父母，导致今生爱情不顺利，弟子真诚求忏悔。弟子遇到佛法后，明了因果，自己造孽，自受果报。弟子以真诚心向观世音菩萨求忏悔，向冤亲债主求忏悔。求菩萨慈悲，赐我一个幸福的姻缘，共同孝养父母，弘扬佛法。

希望他能满足：（可重写自己的择偶预期，只要条件提得合适，发愿能够践行，则善愿必能实现。）
1．真修行，真学佛，恪守五戒，共建佛化家庭。
2．人老实，真诚，孝顺父母。
3．工作稳定。
4．家庭条件和我差不多就行。
5．长相和身高能让我满意就行。

为此，我（起码应该）发愿：
1．从今以后不再犯邪淫，断除身口意不净之业。
2．尽自己的努力去劝告别人邪淫的危害。
3．坚持修习一种观-音-法-门，并回向给特定的对象，如，冤亲债主、在过去今生被自己伤害过的人。
4．发愿做善事1000件（或以上）。
5.……

求观世音菩萨的感应可以有很多法门：
1、修持观音菩萨圣号
2、大悲咒
3、六字大明咒
4、般若心经
5、观世音菩萨普门品
6、观世音圆通章
7、《华严经》善财观自在菩萨章，等等。

法门选定一个，不可多选。因为一门深入才会修有所成。求愿迫切的版友不妨到佛学版浏览或提问。对佛菩萨提出的条件要合适（佛菩萨只会帮我们断除贪心欲念，绝不会增长我们的贪心，求姻缘时应向菩萨求个利于人格修养的姻缘），发愿的事要能够完成，那么好姻缘也就不远了。

另，要获得美满的姻缘不可忘记以下几点：
1.多多献花给佛菩萨，求佛菩萨牵引良缘来到。
2.多多参加放生，放生功德很大，放生是很有感应的。
3.多多给人方便，给人家好印象。
4.多多接触人群，参加活动以触缘。
5.切不可犯邪淫，因为这会断情缘。而且就算有缘，也是恶缘不息、善缘不长，所谓怨憎会、爱别离。
6.助印宣传戒淫的善书经典，并劝人不犯邪淫。
7.行功立德，求佛菩萨协助。
8.自己先当个好人，再去找个好人来相爱。因为“物以类聚,人以群分”。
9.此生开始，生生世世都不邪淫。想得好姻缘，一定要发此愿，并且非发不可。
10:多修习自己选定的观-音-法-门，如是日日不退，观世音菩萨会安排好姻缘。

最后，祝各位都有一份称心如意的姻缘，一生安乐，福寿康宁！

附：印光大师法语
人生世间，父母、寿命、相貌、学问、夫妻、儿女，皆是前生所作之业之所感召。
若有大功德，则会过于前生所培。
若有大罪过，则便不及前生所培。是以要认真修持，以转前业也。
汝欲得有学问、有才能、有德气之好丈夫，或恐汝前生未培到这个福，则便不能满汝痴心。
楞严经谓：念观世音菩萨者，求妻得妻，求子得子。求妻于菩萨，谓求菩萨加被，得其贤慧福德之妻。
汝求夫亦然。极力念观世音菩萨，当能满汝所愿。否则，纵得好夫，或者又有短命、多病、祸患等事，况未必能得好夫乎。
汝切不可以貌美起骄傲心。此心不息，便难载福。
汝能谛听我语，则汝之所受用者，皆有超过前生所培多多矣。 （印光大师：与周福渊女士书）

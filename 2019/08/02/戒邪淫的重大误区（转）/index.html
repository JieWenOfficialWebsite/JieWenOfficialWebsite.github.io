戒文~戒邪淫的重大误区（转）

Posted on 2019-08-02
刚刚看了一下我在戒邪淫网上发表的第一篇文章，是在2008年9月份，文章的内容也是我要发誓戒邪淫等等。到如今2012年4月14日，已经三年半的时间过去了，我依然在戒邪淫路上，并且被折磨得欲死欲狂。期间我为了戒邪淫做过很多努力，但由于自己的意志力不坚定，大都半途而废，不过要不是菩萨的加持，我想我的结果比现在还要糟糕。真心感恩佛菩萨。
　　邪淫的恶果，网上已经说得很详细了，反正我手_淫过后，必定和妻子或者岳父岳母大吵恶吵一架。我平时脾气还算不错，性格也较温顺，和家人吵架不多，顶多是言语几句。但有几次特别重大的吵架，几乎都发生在我手_淫后的当天或第二天，至于打麻将输钱，心情沮丧、浑身无力、萎靡不振就是常事了。我深信因果报应，深信佛菩萨慈悲，也在现实生活中深深感受到邪淫给我带来的生不如死的感觉，但为什么这么多年我依然无法戒除手_淫？意淫也很难控制？为什么？为什么？我最后归根于自己的意志力不坚定，发愿不彻底，可是不管怎么分析，我要的最终结果就是戒除手_淫啊。我想很多人和我一样，发愿戒邪淫——挺过几天后——邪淫——痛悔——发愿——再度邪淫——，多次循环后，铁打的汉子也受不了了。
　　但是，我刚刚找到了戒邪淫过程中的重大误区，最起码对我来说，是重大误区，我真心把这个认识和体会发表出来，希望能对我，对别人有作用。下面发表出来以下几点：
　　1、很多人深知手_淫的危害，主观上非常想从此远离手_淫，但生活中，仍然屡次发生，无法控制并且痛苦万分，在心理学中，这具有强迫症的特点。
　　2、因为很多人认为邪淫是可耻的，会损害福报，会让自己事业家庭爱情不如意，甚至会堕地狱（其实事实上也确实如此），在心中自悔自责，给自己脆弱的心跟加大了砝码，给予更多无形的压力。
　　3、具有强迫性质的人，性格本身就有拘谨、犹豫、节俭、谨慎、细心、过分注意细节、好思索、对自己要求过高，过分追求完美的性格特质，在长期的工作、学习、社会生活中，很容易积攒过多的负面情绪和精神压力，并且无法释放。在这个背景下，对于手_淫本身的恐惧，手_淫过后的悔恨，只能让这些负面情绪和压力更进一步增加，从而让自己进入恶性循环。
　　4、手_淫成瘾者，经常在脑海中和手_淫做斗争，斗争一次，强化一次。
　　说了以上这么多，下面谈谈如何远离手_淫。
　　步骤1：从思想上，认识到手_淫是一种不好的习惯，通过自己的努力和正确的方法，是可以克服的，在心理学上，手_淫无法去除是一种强迫神经症的体现，既然是毛病，就一定能解决掉，不要给自己太大的压力。
　　步骤2：认识到远离手_淫是需要时间和过程的，不要苛求一下子彻底戒除手_淫。真的发生了手_淫也不是世纪末日，更大可不必因此绝望。如果将两次手_淫的间隔时间由1天变为2天，就是进步。由2天变为1个星期，2个星期，1个月，就是进步。将手_淫从1个月变为2个月，3个月，就是进步。
　　步骤3：放轻松，去掉所有压力包袱，轻装上阵，将手_淫当成和“感冒”“肚子疼”一样的病症去治疗。
　　步骤4：平时该做什么做什么，尽量不去看色_情文章、A片等等。说实话，就是神仙天天看这个，也得淫-欲炽盛。这个和你吃盐就得口渴喝水是一个道理。
　　步骤5：最重要的一个步骤。在觉得自己想手_淫的前一小段时间，通常会觉得烦躁，不安，类似抽大烟的那些症状。这个时候告诉自己，这不过是正常的心理生理反应，是病症的一种表现形式，我不必在意。例如用一个定时闹钟，让它每3分钟响一次，钟响时大声说“停!”予以驱除。多次做到后，便依次改用正常声音、微弱声音直至仅在内心说“停!”来驱除强迫观念等强迫症状。 如果觉得这样仍然无法减轻，可以告诉自己，手_淫是幼稚的，不合理的，
　　步骤6：带着“不安”做事情。顺其自然，为所当为。慢慢的，这份不安会消失。

邪淫使我走上学佛路

我出生在北方一个普通的农村家庭，父亲是一位小学教师，母亲务农。现在的我在一所普通本科大学读书，即将毕业。儿时的记忆依然清晰，仿佛就在昨天。如今，我已皈依佛门，发愿生生世世跟随诸佛学，敬遵三宝教诲，追随佛菩萨的脚步，一心求生西方，永脱轮回。我虽学佛不久，也不敢称自己是末学，但我很想把我的这段经历以及我的心得和体会和大家分享，愿更多的有缘人能从中有所感悟，能够坚定信愿，勤修正法，断除诸恶，业障消除，究竟安乐。

也许是过去生中累积的淫-欲习气太重，记得我还在上小学的时候就已经对异性有了爱意，看到电视里漂亮的女演员就会念念不忘。后来，违缘现前，在邻居家一个跟我同岁的女孩的引诱下，我们偷偷玩起了“过家家”的游戏，有时模仿电视里的镜头去接吻。类似的事也发生在我和同村的其他女孩身上。到三年级的时候，长大了点，也稍微懂事了点，觉得这样的事是不该去做的，就再也没有发生过那些事了。可是记忆永远存在，每每想起，我就很羞愧，一个小孩做这样的事，真是业力甚大，习气驱之所为。由此可见，电视和网络对人的影响非常之大，凡夫有累生累世的习气，一遇到恶缘便不能控制，随顺贪欲和习气做出一些不该做的，一些极为丑陋的事，一些畜生才做的事。当今社会，电视、网络和杂志里宣传的内容充满着杀盗淫妄，使多少人沉迷，使多少人堕落，它们对社会的危害实在太大太大了。我们明白的人一定要努力克制自己，远离那些不好的、不善的、丑陋不堪的东西，守护我们纯真善良的本性，勿使自己堕入欲望的深渊，轮回的苦海。

我小时候虽做过这些可耻的事，但我还算是个懂事的孩子，知道父母很辛苦，所以自己在学校也好好学习，成绩都是数一数二的。初中三年没有什么出格的事，每学期都是一心想着考个好成绩的。直到高中，噩梦开始了。高中第一学期，我的成绩突飞猛进，老师很高兴，我也很欢喜。可就在高一第二学期的某一天，无意间，我学会了手_淫！后来，一发不可收拾，开始了我愚痴堕落的生涯。上瘾的东西它总会按时在脑海中想起，几乎每天都沉迷于此而不能自拔。学习成绩开始下降，相貌开始丑陋，种种不如意的事让我苦闷。高考那年，我成绩很低很低，没有考上大学，又不甘心失败，便选择了复读。第一年复读，依然是在噩梦中度过，恶习不改，再加上轻微的心理强迫症，学习没有丝毫的进步，复读以失败告终。两次高考失败，我以为我的人生就这样失败了，我很灰心，很沮丧。幸运的是，我的母亲，一个深明大义的坚强女性，坚定地要我再次踏上复读的旅程，因为她相信她的儿子不比别人差，她相信她的儿子肯定会有出息的！那时，我在内心深处还是不甘心就这样输给命运，我又一次复读了。这一次，我选择了另一所学校，换个环境，不认识的人多，心理上也轻松一些，再者，在这所学校复读的学生学习不是特别突出，我在那儿学习，成绩还算是进入优秀的一列了。这样，渐渐地，自信心生起了，学习开始进步了。可是，恶习依然在身！难改啊，真的真的太难了！多少次，我曾深深地责备自己，我为我的行为感到羞愧，我也认识到自己太邪恶了，太不是人了，可是，奈何我愚痴至极，智慧不够，根本不知道该怎么去解决，甚至连邪淫到底是好是坏都不知道！我那时候处在极为矛盾的状态，很可惜，因缘不成熟，没有人来告诉我对错，没有人来教导我怎么去做。我在网上搜心理健康教育，得到的都是“适当手_淫无害”。苍天啊，大地啊，我是凡夫啊，我能控制得住自己吗？我能适当得了吗？

说它是对的，可它为什么那么丑陋、邪恶，要偷偷去做！意淫自己喜欢的任何对象，亲人，朋友，同学，陌生人……我的姐妹，她们是我的亲人啊，我怎么可以干这种事！说它不对，可大多数同学都说这是正常的生理需求，又不是真的去做那些事，很正常的。我处在这两种认识中，愚痴、迷茫地度过了第二个复读年。期间，或许是我过去生中的善根萌发，内心深处总觉得邪淫是不对的，我要努力去改。偶尔有一次，我在QQ空间相册里传了一张本师释迦牟尼佛的像，那时候虽未信佛，也不懂佛教的实质，但希望佛能给我带来好运，让我早日戒除恶习。很遗憾，直到参加高考的前一天晚上，我都在犯邪淫。那次高考，我自己评估，按自己的能力终将又是一次失败的高考，却不曾想到，那年的考卷题目很适合我，尤其是生物这门课，我不会的基本没有出题，大多是填空题，都是记忆型的，好在我平日还算态度端正，很认真地记了，那些我都会。满分72分，我估了65分左右。其他科目也正常发挥，这样，我的总分成绩比往年高考增长了一百多分，我的分数高出了重点分数线，我报了一所普通本科学校的一本专业，我被录取了！

在这里很想提到的是，长期邪淫的人很容易患心理强迫症，因为他长期处于矛盾、自责的状态，犯的时候因为贪心驱使不顾后果，犯了之后又深深悔恨，这样反反复复，心理很容易形成一种惯性，他自己再也没法控制自己，也不会相信自己。他心里很清楚这样做不好，可就是摆脱不了束缚，好像大脑被无形的东西控制了一般。

就这样，我进入了大学，开始我的大学生涯。不好的习惯还是随身携带着。在今天这样的条件下，电脑、MP4等设备给身染邪淫恶习的人提供了莫大的方便。网络上，你想看什么有什么，你就顺着自己的习性与爱好，去寻找，去享乐……我在大学期间修学过一门课程----大学生心理健康教育。给我们上课的是一位年轻的女老师。有一天，她讲到性心理方面的内容时说自慰是正常的……在这样错误的指导下，我看污秽的电影时，心里的罪恶感没有了。老师不都说了嘛！愚痴啊，愚痴太可怕了！现今社会，不知有多少所谓的专家在宣扬着错误的观念----手_淫无害。

多少次，我想改，却改不了！一旦上瘾，再戒就难了！我是个凡夫啊，六道里面的愚痴众生！我所面对的敌人太强大了，多少次，我都失败了！相信必是祖宗有德，父母善良修福，感得三宝加持，2012年那个寒假，我用手机百度了“戒色”两个字……浏览到“不净观网”，看到了慈悲的净空老法师讲的“学佛问答”的视频，慢慢地，一步一步地，我开始了解佛法，认识佛法。毕竟，我心中很想知道这些事情的真相，因为在我们村子里也发生过很多奇事，例如十三岁的小孩给人算命啊，或是某个人见鬼了，我很好奇，很想知道这事是不是真的，到底是怎么一回事！

就在那年冬天，我回到学校开始用电脑在网上搜索与因果报应相关的事例。刚开始搜到文章，以文字的形式记载事件，看完后并不能使我全信。后来，很幸运地搜到了海城大悲古寺录制的因果报应视频，一例例真实的报应，一个个惨痛的教训，让我对佛法生起了坚定的信心。我也明白了邪淫是错误的，淫-欲是可以去除的。因果不空啊，造恶是要遭到报应的。我又开始了戒邪淫，下决心要彻底解除它！奈何凡夫根性太强，习性难除，每次坚持十天或半个月就犯一次。好在我认识到，这跟以前相比，我已经进步了，至少对身体的伤害降低了一些。就这样，改改犯犯地，到了2013年9月，我已到大四，刚开学就开始着手找工作了。我见有很多同修向佛菩萨求工作、求姻缘都如愿了，为了能找到满意的工作，我也学着写了一份求工作疏，向大慈大悲的地藏王菩萨求。签工作前，我信愿不坚定，习气又现前，就在我签工作的前一天，我心生烦恼又犯邪淫。9月15日那天，我签上了满意的工作，我深深的感受到一定是地藏王菩萨的加持，若非菩萨慈悲加持，我是不会这么顺利找到工作的。我的成绩太低太低了，低得只要是见了的人都会摇头。我第一次见到签我的那个人就感觉自己有希望了。那天他在住的楼下为我们前去应聘的同学讲公司的待遇等等情况，特别特别真诚。我注意到他的右手腕上戴一串佛珠，脖子里吊着弥勒菩萨的玉坠，我心里想他肯定信佛。后来才知道他并不信佛，他看到我简历中有一栏写“宗教信仰----佛教”，他还问我我有没有加入什么组织，我回答说没有，只是让我向善的方向发展。把简历交给他，我们就离开了，他说要我们回去等通知，什么时候打电话什么时候过来面试。我一心认定了这个单位，那一天我穿着西服，手里拿着手机一直在等电话，可是到了下午四点也没有人打电话来。四点半左右，我过去找他，看到他正在面试其他学生，我便在旁边等候。轮到我的时候，他跟我说：“我今天中午跟你们学院的领导一起吃了饭，也将你们的简历拿过去让他们看了，排了序。你的成绩实在有些低，我不敢签，你们的陈老师在我面前说了你好多好话。说实在的，我也觉得跟你很投缘，我个人也很欣赏你，就是这个成绩，我还需要跟我们领导申请一下，我在领导那边替你再争取一下，你自己也要努力……”听到他讲的这些话，我感动呐，回宿舍的路上，我回想着他说的每一句话，惭愧的心生起了。我们的副主任陈老师跟我接触并不多，他也只是通过我们的班主任老师了解一下我，他为什么要帮我说好话，用人单位的这个人我一点都不认识他，他为什么要替我在公司领导面前申请！我深深地惭愧，一路上，泪花直在我的眼眶里打转。回到宿舍，就我一个人，我狠狠地抽自己嘴巴子，我羞愧，我对不起这些真诚帮助我的人，对不起父母的养育，对不起老师对我的理解，对不起陌生的招聘人……我跪在地藏菩萨和观音菩萨像前痛哭流泪。我发誓，今后，一定要彻彻底底地把邪淫戒除！永远永远都不会再犯！就在我刚换下西服的时候，同学给我打电话说用人单位要跟我签，让我带协议过去。我便又穿上西服拿着协议去了。这个人跟我说了好多鼓励的话，我也向他表了态，要好好学习提升能力，好好工作回报帮助过我的每一个人。离开时，我深深向他鞠了一个躬，感恩他对我的赏识！工作签了，第二天早上，我又穿上西服，去跟我的班主任老师、替我说话的陈老师，还有曾鼓励过我的女老师一一鞠躬表达感谢！

就这样，签了一次工作，父母和老师对我的帮助真正地把我感动了，我誓死也要彻底戒除邪淫，要做个善良的人，我要回报所有帮助过我的人。9月15日签上工作直到今天，说句真心话，我真的从心底里不想再犯邪淫了。有时偶尔会想起那种念头，一想到我就感觉很丑陋，很邪恶，很肮脏，很愚痴，再一想到我的父母，我的老师，诸佛菩萨……宁肯去死，我也不会干那么缺德的事！我披着人皮，长着人心，我要做人！诚能感动上天，诚能感化世人，我被父母和老师的恩德深深感动，我相信自己，今后我永远永远不会再犯邪淫了。我心里想的是阿弥陀佛，装的是我的父母和老师，我再也不会去做那些愚痴的事了！

邪淫让我走上了学佛路，我找到了一条光明大道，一条智慧的道路，一条可以解脱的道路。我也是因祸得福，所以还是要感谢邪淫引我走上学佛路！

现在，我的身心清净了好多，我的生活方方面面也渐渐好转，我相信自己一天天会好起来的，佛菩萨的教诲我也牢记在心。就在前几天，我不认识的同学因为心情不好，而我恰好路过看了他一眼，他就追过来打我，此事让学院知道了，给了他处分，影响到他正常毕业拿毕业证。他当着学院老师的面给我道歉，我告诉他，我学佛修行，不想跟任何人结怨，我不需要人跟我道歉，我也不需要去原谅谁，那些不好的人和事我是不会放在心上去污染我的清净心！我问老师这个处分能不能不给，老师告诉我制度对每个同学都是严格要求的，但可以将我的意见转达给学院领导，再看学院领导如何决定了。我尽的心尽到了，剩下的就随缘吧，各人因果自负！

想想这事也挺搞笑的。被打的那个晚上我回到宿舍跟舍友说了一下，我自己不当回事了，可舍友咽不下去这口气，他们要找人去算账，我劝都劝不住。好在那个小伙子那天打完人回去了，不住在宿舍，否则就要连累舍友了。净空老法师曾说佛的福报很大，周围有很多护法神，佛可以原谅你的恶行，但护法神不允许你胡来。各人心量大小不一样！出世法跟世间法是一样的，是一不是二！

父母养育我，老师教导我，我愿用我的这份真诚去感动上天，感动佛菩萨，愿我生生世世的父母、老师以及往昔我所伤害的一切众生，皆能因我的改过、断恶修善之因，早日蒙三宝加持，业障消除，现世安乐，命终蒙佛接引，往生西方，永脱轮回苦海！

惭愧弟子也向我所伤害过的一切众生忏悔，我因愚痴伤害你们，真诚祈求你们能够原谅我的罪行，我真的错了，我不该伤害你们的，我愿自今日始，尽未来际，勤修正法，断除诸恶，愿将我所修的一切善行、功德回向给你们，愿你们在三宝的加持下，业障消除，早日往生西方极乐世界，出离苦海！等我命终，往生西方，彼时，我们都是清醒的人，觉悟的人，我们谁都不会再伤害谁，我们是阿弥陀佛的好学生，我们是好伴侣！

惭愧弟子为自己以及我所伤害的一切众生，与我有缘的一切众生，尽虚空遍法界一切众生，今在佛前真诚求忏悔：

往昔所造诸恶业，皆由无始贪嗔痴，从身语意之所生，今对佛前求忏悔！

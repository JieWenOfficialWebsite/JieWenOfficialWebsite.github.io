戒文~如何才能找到让你幸福的人

如何才能找到让你幸福的人

　　对我们能够给幸福的人就是自己心里能够接受的人，是自己喜欢的人，也是自己最爱的人，比如，能够给菩萨能量的人就是众生，因为菩萨非常爱众生，为众生愿意付出一切的代价，你愿意为他付出一切代价的人，才能增添你的能量，为他付出，达到目的时，你觉得非常幸福及高兴，觉得自己为他所做都是应该,会有殊胜的一种感觉。这个人才是给你拥有正面能量的人，你对他的很多事情充满好奇，无论遇到什么样的事都会想到他，你愿意为他做一切，你会发现那个人的能量很大，值得用罄一生去不断尝试，值得为他付出，这也是一种幸福的来源。

　　幸福需要安全感，没有安全感，不可能有幸福感，所以你千万不要找一个没有安全感的人过日子，没有安全感的人，他不会有让你幸福的能量，这种人一直在排查可能的不幸和焦虑未来的灾难。他会一直在想将来该怎么办，跟别人怎么办等，一直担心祸事即将降临。他每天扑向那些或有或无、或虚或实的灾情，不停算计、紧张和忧愁，他活也就像死牢里的囚犯一般，每天活在恐惧的圈套里，他没有办法给你幸福感，更没有能力给你正面的能量。

　　幸福的根本是乐观和信任。拥有正面幸福能量的人，会对生活乐观，也对自己有信任。大家都知道生活本来就悲喜交加，所以，我们要学会坦然面对社会，面对生活，当你遇到快乐时，你要学会尽情享受，当你遇到烦恼及苦难时，你要学会理性解决问题。我们要相信自己，自己定胜天地，自己才是真正自己的救星人。我们人生之中风风雨雨，会遇到真正确实无法获胜时，我们要学会坦然地去接受。他们能够正确认识自己，有自知之明，不会自我贬损也不会自我膨胀，他们在该独立的时候独立，该求助的时候求助。我们千万不能把自己当成万能神，也不能把自己当成窝囊废。这样才能你内心幸福的能量释放出来。乐观和自信后面，还深藏着对人生的豁达与包容。

　　幸福的伙伴是豁达与包容。大家都知道，你跟一个心胸狭窄，或者斤斤计较的人在一起，不可能有幸福感。所以，不要找一个无知的人过日子，也不要找不懂别人心的人过生活，你会痛苦的，因为，他们没有树立起完整的人生观，或者对事情价值的判断缺乏基准线。他们常会做出匪夷所思的决定，不能独立思考或者过于固执己见。他们优柔寡断或专横无礼，他们扭捏作态或者刻板无情。不是因为别的，正是因为无知。

　　他们不知自己错在哪里，总是认为自己所作所为都是对的，别人所作所为都是错的，还有强词夺理，斤斤计较，你跟他就像管在看所室一样，他不完不了的问题就像审问一般，你根本就没有自己的空间，你哪来的幸福和正面的能量。豁达及包容人的能量非常强大，而且他们拥有大智慧，他们分得清世界的黑白曲直，不会在人生的道路上跑偏也不会随波逐流。也不会扭曲事物的本质，更不会夸大事情的不利面。他们知道世界运作的原理，明白人人都有阴晴阳缺，他们的心胸宽阔，知识渊博，智慧无量，能量无限。他们在你需要时给你最中肯的建议，

　　有原则却又求新求变，有主见却又听得进劝，而且给你提供如何才能获得幸福之根，什么样的人能够给你幸福的能量，如何拥有正面的能量。

　　幸福不是放弃，而是放下。放弃一切也不一定会幸福，但放下一切的时候，你会幸福的。所以，我们要分清放下和放弃的区别，不要混乱一起。我们不能找一个容易放弃的人过日子，也不要找一个执着纠缠的人过生活。我们只能得过且过永久性地安于现状，才能有幸福感。

　　真正的幸福离不开信仰，信仰是一种梦想，是精神上的追求。世上很多人说，我没有信仰，认为自己很了不起，实际上，这种人最可恶，最可怜的人，因为，他们的精神一片空虚，也不知道幸福到底是什么，也不知道幸福在何处，更不会享受幸福。因为他们没有信仰，也没有梦想。

　　他们遇到挫折的第一反应和最终反应都是逃避，为了抵挡失败或者因为怕麻烦，他们可以放弃一整个世界。他们遇到有一点危险时，第一出卖国家和亲朋好友，第二出卖家人，第三出卖自己。拥有正面能量的人，或者追求幸福的人，坚定自己的信念，拥有人生的目标，知道自己的所需并为之不断努力。他们欢迎变化也制造进步。当困难来临，他们不嫌麻烦或贪图安逸，他们知道山丘后面会有道更美丽的风景。当他们生活荣华富贵来临，他们不傲慢，或不自大，他们知道山以外还有高山，人以外还有高人。他们很低调，很谦虚，他们会享受真正的幸福。

　　幸福来自于能量，能量来自于爱，所以我们应该去爱一个拥有正面能量的人，你才能获得真正的幸福，经常有人问我：”活佛，那你爱谁？”我经常回答：”我爱三个人，第一，我爱有恩的人（包括父母，师父，佛菩萨等）。第二，我爱害我的人（仇人，敌人，诽谤我的人，妖魔及折磨者），第三，陌生人。”我每次跟他们，我才发现我自己为什么要活下去，我爱他们每个人的时候，才感觉到人生的价值，活下去的意义，才会让我觉得世界色彩斑斓。

　　他们真的给你了很多很多的惊喜，也给了很多很多的知识和智慧，同时我也感悟很多道理。有人带我弯路，有人带我正路，最终他们让我带进正路，戒断所有扭曲的价值观。虽然我在弯路上遇到很多歪门邪道，但我也体会到真正的歪门邪道是什么，真正的正面悟道是什么，如果没有他们，我们本身就不是一个拥有足够正面能量的人，而且独自的能量非常微小及微弱，所以我们一定要爱一个拥有正面能量的人。这里讲的爱并不是男女爱情的爱，而是内心崇拜及至高往上的人。

　　比如说，具有正面能量的人会把你的人生改善完美，另一个同样具有负面能量的人会把你的人生拖垮，不同空间的畸形与病态会让你过得一团糟，另一个环境优美的空气会让过得幸福自在。因此，我们依靠具有正面能量的人是非常重要，他能够导正你的灵魂和行为，潜移默化中，你会变得更加开朗和幸福。这一定，比任何财富更能长久的滋养你的心灵。

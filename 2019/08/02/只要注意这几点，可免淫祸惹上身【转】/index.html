戒文~只要注意这几点，可免淫祸惹上身【转】

Posted on 2019-08-02
「淫业到头终有报，只是来早与来迟」。这句话看起来好像警示的作用不够深切，就因为如此，所以在世人的心念上，起不了什么惕励的效果，其实淫业之报，经常在无形的进行着，如同有些人因为犯了淫业，减福消禄，或是因而身染重疾，无法医治，甚或绝断后代，或有者终日精神萎靡，影响己身事业前程者，这都是渐次的加以无形的报应，不得不谨慎警戒自己。



俗话说：「年少不会想，老来不象样」。这句话虽指着青年不懂事，不知上进而言。其实这句俗话，用在戒淫方面也是甚为恰当，因如果年轻时不懂事，在淫方面不加节制，纵己情欲伤身害体，年老时，必是未老先衰，体弱多病，到时哀叹悔恨均已迟矣，何不在日常生活中多小心戒慎自己，好好的化淫念为正心，以成己之德。
在此提供一些在造成或是阻止淫念上几点看法，供大家参考。
在男人方面：
1. 不可经常心存淫思、淫-欲，想占女人之便宜，不论在言语上或行为上，都得秉持正念而为，千万不能淫言色语挑逗女人，在此则必造下罪业，由于现今社会之开放，此种现象特别严重，当思戒之。
2. 不看淫书、淫画、黄色影片，或不当不良之杂志，因一切淫念之起都在于此，看了之后想入非非，那淫业则随之而来，不接触不当的各项色_情之物，就能正心、正德。
3. 不涉足不当之色_情场所，因机会多，造罪亦多，不去不当之场所，就没有机会造淫业了，可依个人喜爱在正当的休闲活动求发展，或是学佛、习道，不只可以正德，更无罪可造也。
4. 虽既已婚，夫妻敦伦，亦得节制保身，并非夫妻就可纵欲，因此亦会伤及双方身体，故亦得戒慎之。
在女人方面：
1. 端正固有道德之观念，由于现今社会之西化，固有道德渐次没落已久矣，所以也造成现今社会妇女不遵守伦理道德的原因，故身为女人，虽社会非常的糜烂，更应该负起转化社会风气之责任来，不随风而流，应正风而行。
2. 在穿着上要特别注意，如果为了崇尚时髦，穿着过于曝露，易引起男人不当的心念，不但己身造罪，亦因而引导他人造业，能不慎吗？
3. 言语当正念严厉，尤其与男人谈话更得注意，不可轻佻，不可随便使得男人以为妳可以随意而求，造成淫业之来，如有贪欲之男人想占便宜，得正言严厉斥之，以免误导而造罪业，多多在正业或相夫教子上下功夫，则必无闲暇去思想不当的作为，因一有空想，必图享乐，一有享乐之念，则必与淫业有关，不得不加思警惕，尽量使自己能适切的正当生活，则妇德可成就也。
以上有关男女之戒淫项目，只提大概供现世男女参研，能依这几项慎守之，淫业则不会有之，盼大家齐予互相勉励。

戒文~邪淫果报现前，真心求忏悔！

Posted on 2019-08-03

我今年32岁，在上海工作。07年毕业至2010年，想要找一段稳定的感情，但一直不顺，期间就和数人暧昧并发生关系。2010年底换了一个工作，并通过一个项目认识了一个男人，有妻室，一对儿女，当时他们的夫妻关系正处在冷战，准备离婚。我和他几乎是一见钟情，因为异地，每个月我们见面六七天，我成了传说中的“小三”。中间他和妻子协议离婚，对方不同意，但他是一个善良的人，舍不得伤害小孩，不愿意走诉讼程序，就这样，坚持到2012年中，持续了一年半的时间，他突然就提出分手了。到现在，时隔10个月了，我还是没法走出这段阴影。
接下来说一下，我因为邪淫而遭受到的果报：
1. 自从和他在一起后，我的工作状态直线向下，脑袋没有一天是清爽的，总是昏昏沉沉，直到2012年7月公司解雇了我，现在10个月过去了，依旧没有找到合适的工作；
2. 父母在2012年上半年相继生病住院；
3. 自己的身体变差，总觉得没有力气，脸色灰暗，没有一点光泽
4. 个人感情一直不顺，我想这也是这么多年邪淫的结果
很庆幸通过朋友介绍，开始学佛，知道自己造了很深的孽。想想自己也是还有一个哥哥，我们兄妹感情那么好，我又怎么可以忍心伤害他的两个小孩。现在开始定期助印、放生、诵经。
想想自己平时脾气差，耐心差，时时给父母脸色，也是大不孝。
在这真心求忏悔：往昔所造诸恶业，皆由无始贪嗔痴，从身语意之所生，皆向佛前求忏悔。
发誓彻底戒掉邪淫，希望我的亲身经历，可以给现在还在邪淫的人一些启示，望大家早日悔改。
百善孝为先，以后要好好孝敬父母！善待身边的每一个人

戒文~冲出色欲的毒潭[终结篇]

Posted on 2019-08-03
　　万古以来，人世间从来没有人能堕落到色欲的深渊后还能重新爬出来。惟有今天，在当今圣者不离不弃的慈悲牵引下，并经过在长达十年的巨大的魔难中奋力精进，吃尽人间的无数苦难，如今才得以脱身无底的色欲的毒潭。看清了色欲的根本面目后，觉的有必要让更多的人也了解它，从而使更多的人也能解脱色欲的苦苦折魔，重新堂堂正正挺立于天地间做真正和美好的人。
　　天理，天道（也叫天法），听起来好象是概念的东西，但却是最真切的存在。人生存的空间，只是人的眼睛能直接看到的分子细胞这一层次的空间。而我们的空间又是密切联系着更微观的微生物、病毒等人“看不到，摸不着，但客观存在的，而又能实实在在反映到我们的这物质空间”（圣者语）的微观空间的生命体。那个空间世界，就是由天理在主宰一切。
　　人的每一个念头都有更深空间的来源。人符合天理的念头（也叫正念），是来自人身体中善良本性的生命体；而人的每一个不好的念头（也叫邪念），却是来自无数不好的生命体。那些不好的生命体可以表现为鬼和兽，更微观就表现为鬼和兽身上的各种各样的微生物——微观下的虫子。他们存在于我们的身体的每个细胞特别是神经细胞中。人身体饿和色的欲望的感觉和思维，，在某种程度上可以说是这些微生物——微观下的虫子，它们的集体意识和生命活动信息在身体表面的表现。所以大家看《西游记》，孙悟空有瞌睡虫可以主宰人，唐僧和猪八戒的身体的区别是一个洁净一个肮脏，直接说就是身体细胞和神经细胞中微观的虫子多和少的问题，所以他们一个六根清净，接近佛身，一个欲望满身，接近猪身。大家都看到了猪肉是会生虫子的，人身体也是有虫子的，从皮肤表面到肉到五脏六腑，虫子小，就在细胞中，眼睛看不到，虫子大，就在身体中。就如《三国志》中记载，神医华佗给许多人看病，许多人是把他们身体里的大虫子给排出来，病就好了，各种形状的虫子都有。最能体现虫子的意识状态的就是，许多吸毒者后来毒瘾发作时，感觉如万虫咬身般的痛苦，正常人为什么没有这中感觉？那是真的有虫子在咬呀，只是西医看不到而已。
　　吸毒的人为什么会招来无数的虫子，正常的人身上为什么就没有？因为天理（天法）在维护着世界的秩序和规律。世界分阴阳，生命分六道，不同的道被天理的更微观粒子构成的铜墙铁壁一样隔离开来，就象无菌的隔离病房一样造就一个纯净的空间，就象屋子窗户的窗纱一样保护人类不受蚊虫的叮咬。人身处于人的道中，天理在保护着人，防护着那些恶毒的生命体，不让它们有机会接近人的身体。所以表面上看起来，我们人类和虫子、病毒在一起，实际中间却是被无形微观而真实存在的天理的更微观粒子构成的铁墙隔离开的。天理在维护和保持着人类的生存和安危。但是人的心念是一把开启一切空间的铁墙大门的钥匙。正念开启的是圣洁世界的铁墙大门，而邪念开启的却是肮脏世界的铁墙大门。吸毒的人开始时候为了享受飘飘欲仙的美好感觉，为了享受意识脱离身体束缚的轻松体验，不能走正道，走修炼道德的大道，却向外去求，借助毒药的功能，所以只能得到暂时的美好和轻松。更严重的是他们明知是毒药，还要喝，那么这一个邪恶的意念就开启了肮脏世界的铁门，结果肮脏世界的无数可怕的微观的虫子就降临了他的身体的细胞和神经细胞中。从此他只能靠毒药来继续喂养那无数的虫子，虫子来时，吸了毒，虫子吸到毒汁，就满足的飞走；没毒吸，虫子饿了，就咬他，他就觉的万虫咬身，生不如死。所以圣者才说“好坏出自人的一念，一念之差也会带来不同的后果。”为什么天理这时候就不管不保护他了呢？因为是他自己要的，“你自己要的谁都不管”（圣者语）。
　　而比吸毒更无可救要的却是色欲的毒潭。毒谁都怕，上瘾后绝大多数人马上就后悔了，只是无能为力，他们本性是清醒的。色欲的毒潭，却让人越来越落入无底深渊而自觉其乐无穷。所以自古君主为色所迷，到死不悔不醒。纣王、齐襄公、隋炀帝、唐高宗、唐玄宗、李后主......至于官宦巨富士子百姓因为色欲而丧家陨身，更是不计其数。当今世界，美前总统克林顿、意现总理贝卢斯科尼、公开自己同性恋身份的议员和影星，同.性.恋和乱伦正日渐泛滥的中国和世界，还有无数的包二奶的官员和色情娱乐场所。色欲的阴风正扫荡世界。疯狂大潮流中能醒悟的不多，醒悟后能摆脱的更是登天之难。色欲，如温水煮青蛙，杀人于无形，让人死而不悟。它迷惑人的手段是：色欲中的紧张，激动，刺激，兴奋和欢喜。
　　圣者说，“人就追求精神刺激，怎么解渴怎么来。”谁都以为能叫人紧张，激动，刺激，兴奋和欢喜的东西就是好东西，所以只要能带来紧张，激动，刺激，兴奋和欢喜的东西就不断去看，去听，去摸，去做，去想。其中以通过想像而获取刺激和兴奋最普遍和常见，特别是心中还有良心的人。生生世世轮回中因为误入色欲后形成的紧张、兴奋欢喜的那一个念头，显现在今生小的时候或成长过程中，看到和听到、接触到某东西，突然就感受到自己的身体和心中一种触电般的兴奋和快感。从此就以为这就是人活着的最美好的东西和经验感受，从而以后的日日夜夜里都在用各种见不得光或阴暗的心理在追求和感受这其中紧张和兴奋。但是谁又能明白和想的到，欢喜心是最迷惑人的一颗坏心，和人的善良本性完全背道而驰；谁又能明白和想的到，追求欢喜、兴奋的心将在未来的人生路上把人彻底导向无底的色欲毒潭，从此不断伤害自己和爱自己与自己爱的人，心灵和生活越来越黑暗与低能；导向人生路的坎坷与灾祸，卑贱痛苦，暗无天日而无能为力。以至于后来要爬出来时，无比的艰难和痛苦，悔不当初，深深的哭泣，只恨不能早得恩师的教诲而沦落为不忠不孝之人。
　　那紧张、刺激兴奋的念头和感觉从哪里来？就是那一瞬间无数的虫子苏醒和显现在我们身体的细胞和神经细胞中。它们看到，闻到，接近到了自己喜欢吃的肉体和液体的气味，所以兴奋的发狂。大家可以看一看当蚂蚁们找到一只大肉虫时无比激动兴奋的景象，也可以看一看蚊子闻到血腥味时的发狂景象，也可以看一看苍蝇叮着一块腐肉时的贪婪的景象。当无数微观下的虫子苏醒和显现在我们身体中时，它们的欢喜、兴奋、发狂的情绪和思维就在我们的身体实实在显现出来。于是当人的心灵和身体感受到那强大的兴奋和激动的快感时，人就觉的那触电般的快感。一旦人喜欢这种感觉，并追求它，那么每一次人体会到强大的兴奋和激动的快感时，就是不断在开启肮脏世界的大铁门，而那肮脏世界的无数微观的虫子就不断过来，落到我们的身体细胞和神经细胞中，每一次兴奋和激动，人的身体中的微观下的虫子就一次次多起来。
　　本来开始的时候，人是很长时间才能感受到一次触电般紧张激动兴奋的欢喜，因为虫子的数量少。可是随着人越追求它，要它，它们就越来越多和强大，越来越活动频繁起来。到最后，人一想要它，它马上就在身体表面显现出来。在到最后，你不要它，它依旧不受你控制的显现在你的身体表面，这时候，人就完全被它们给控制了。这时，你看到，听到，接触到的任何东西，都可能成为它们显现出来的理由。人已经对它们无能为力了，人的身体和思想完全被这肮脏低能的东西主宰，被无休止追求紧张、刺激、兴奋、欢喜的虚幻感觉控制。这时候，人活着就是为了它们，或者说，它们成了人身体和大脑的主宰；这时候，人的智慧、清醒的大脑、美好的心灵、健康的身体、和光明磊落的人格魅力，就会被它们吞噬和掩埋到黑暗的垃圾粪坑中了。理性的丧失伴随的是一切美好的东西都将远离和丧失，伴随的只有肮脏卑贱、可悲和无可奈何。
　　追求紧张、刺激、兴奋的欢喜心最终把人带入万丈无底的色欲深渊。让人在那最黑暗的世界中绝望的永远后悔。被无数微观下的虫子变成一条“可怜虫”。想象一下，一个满身是虫子的人，走到哪里会受人尊敬和爱戴与珍惜呢？天地都会嫌弃他的。
　　所以，保持一个纯正的正常的心念对人来讲是最重要的严肃的事情。古代的人心中有善念，知道和相信“一正压百邪”、“举头三尺有神明”、“暗室亏心，神目如电”、“要慎独”。他们正是保证自己永远身处天理给人划定的“人道”的安全和洁净的空间世界中呀。否则，邪念缠身，就会给自己招来灾难和痛苦，招来低灵和可怕的生命体呀。到后来，就是失控了，不能自己主宰自己，导致身在深渊中。那些宣扬无神论的人，最终就会偏离“人道”，他们正是把人类往万劫不复的深渊中推呀。
　　一旦人已经陷在色欲的深渊中，后悔了，可是陷的太深，靠自己的力量已经无能为力了。该怎么办？出路在哪里？
　　这里给出几个建议：
　　1. 要找到当今的圣人的教诲。因为只有真正把道理讲明白，并让道理的每句话都变成神妙和灵验的真言，这样的教诲才是有无边力量的。当一个人被无数层层如山的毒虫掩埋和影响时，如果没有神妙的力量帮你清除它们，光靠嘴皮子，人依旧在痛苦之中。
　　2. 要坚定和能吃苦。持之以恒才能功成，道理不多说。吃苦，难呀。“修心断欲去执著”，那剜心透骨的难以割舍，把垃圾当黄金的死死不放，太难了。但是只要决心放弃那看似迷人的毒物，最后必将成功。
　　3. 要苦修。陷落的越深，越对自己要严格。有的人不吃肉不算，到最后连盐都断了。普通人身体是靠盐来支持的，一旦长期不吃盐，一般的人是承受不了的。他们这么做的目地和道理，可能是那些不好的念头都是在吃饱喝足油和盐做的食物的身体中产生的，那不好的生命体也适应盐的身体环境，为了扼（或饿）死那些毒物，在特殊的时期采取的一种极端的做法。杀敌一万，自损三千嘛。办法虽低级，不能推广，却是对色欲重的人最好的解脱的方法。
　　4. 要多看别人的长处，比较对照自己的不足。
　　结束语：色欲毒潭的可怕，是超出人想象的。冲出色欲的毒潭，是古人从没做过的。如今，这一切在圣者无边的智慧和法力的慈悲中，终于成就了现实。当人走出色欲的毒潭，健康、自信、智慧、和心灵的澄净美好就会常伴左右了。人也才真正知道，生命的快乐和美好。愿普天下的善良的人也都能走出色欲的毒潭。

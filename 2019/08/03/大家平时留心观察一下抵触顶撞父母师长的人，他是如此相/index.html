戒文~大家平时留心观察一下抵触顶撞父母师长的人，他是如此相

Posted on 2019-08-03
　大家平时留心观察一下抵触顶撞父母师长的人，他是如此相
　　平时多多观察, 就知道什么是福禄
　　看看不孝顺父母的人
　　1. 工作如何
　　2. 配偶如何
　　3. 子孙如何
　　4. 职位如何
　　再看一看不抵触顶撞父母师长的人
　　1. 工作是不是顺利
　　2. 受提拔是不是常是他
　　2. 配偶贤惠与否
　　3. 子孙是不是端正聪慧有礼貌
　　4. 他们是不是事业上非常踏实用功, 并且不说人非, 众人是不是拥戴他们.
　　大家知道古代的官员怎么个升官法吗?
　　如果皇帝知道这个官员不孝顺父母, 一票就把他给否决了. 不孝顺父母的人, 不可能尊长, 皇帝当然不要这样的人了.
　　我小时候喜欢读史书, 起初我还认为奸臣都坏透.
　　翻开史书我惊讶的是, 古代的大奸臣, 特别孝顺父母.
　　比如大奸臣严嵩不但对父母很孝顺,对妻子也是忠贞不移,尽管他后来已身居高位,但始终只有欧阳氏一个妻子,两人相亲相爱, 白头偕老
　　比如秦桧虽然受众人唾骂,但在他的爸妈眼里也是个孝顺的儿子, 在他的老婆眼里也是个忠诚的丈夫.
　　比如著名的酷吏张汤, 非常孝顺母亲. 张汤的母亲还在历史上留了一笔
　　......
　　-------------------------------
　　我有时观察天道也疑惑, 因为不管一个人是好人还是坏人, 如果他孝顺父母, 就一定做官, 如果非常孝顺父母, 就做大官.
　　礼敬侍奉父母师长的人, 现世就一定被人礼敬侍奉

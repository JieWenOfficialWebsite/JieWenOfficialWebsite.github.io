戒文~<P><FONT size=3>　　忏悔往昔所有邪淫恶业。<BR>　　本人出差期间在朋友带领下去看

　　因情妇不忠于自己，一男子欲杀死情妇的14岁女儿。2010年5月4日，荥阳市人民法院以故意杀人罪对该男子做出判决。“救命啊，杀人了。”2009年11月6日22时许，荥阳市贾峪镇西南3公里的一个村庄，一声声撕心裂肺的凄厉叫喊声，突然从村民黄杏的家中传出。
　　屋子里，黄杏14岁的女儿兰兰头部血肉模糊，躺在血泊中奄奄一息;而黄杏则披头散发，呆若木鸡地望着女儿。赶来的人们一边救人，一边报警。
　　黄杏反映，凶手是她的情夫刘和平!12月5日，刘和平被抓获，他对犯罪事实供认不讳。
　　2001年，兰兰的父亲和母亲黄杏吵架后离家出走。次年秋天，黄杏认识了在煤矿打工的外地男子刘和平，两人很快发展为情人关系。
　　去年春节后，刘和平从老家回来，发现黄杏与自己的工友双宿双飞，自己也被黄杏疏远。他发誓一定要对不忠于自己的黄杏实施报复。
　　现年44岁的刘和平说，他跟兰兰的关系一向很好。这次下毒手残害兰兰，纯粹是为了报复黄杏的不忠。
　　案发前三天，刘和平到门市部买了斧子和橡胶手套。11月6日夜，刘和平悄悄溜进兰兰的房间，趁其不备，向兰兰头部砸了两斧。认为兰兰已死，刘和平扔下斧子准备骑着院子里的电动车逃跑。
　　因电动车上锁，刘和平返回房间寻找钥匙，看见兰兰未死，又坐起，便拾起斧头又朝兰兰头部砸了一下，随后骑电动车逃离现场。经鉴定，兰兰头部的损伤程度为重伤。
　　2010年5月4日，荥阳市人民法院以故意杀人罪判处刘和平有期徒刑11年，并附带民事赔偿8222.95元。

戒文~男孩自慰过度患上功能性勃起障

　　就像这炎热的天气，人的热情也随着高涨起来。这高涨的还不光是热情，性腺也不例外。但是很多事情都不是多多益善，纵欲过度也是一样。
　　案例收集医院：
　　福州台江医院
　　几天前，小王为了自己的难言之隐来到医院，结结巴巴倒出了自己的烦心事——阳痿。尽管他才不到20岁。刚开始，大夫还以为他自己判断错误，可是小王坦白道，他2年前就已经有了性生活。
　　因为年轻，身体好，而且觉得新鲜好奇，小王的性生活非常频繁，基本上每天1次。
　　前不久，小王和女友分手了，百无聊赖，他迷上了上网打发时间。没想到，从迷上上网之后，小王沉溺于一些小电影和跟一些女网友聊天。由此，他向大夫坦承，自己经常在电脑前面自慰，有时候甚至还请假不上班，几乎一天都坐在电脑前。
　　他跟大夫说，5月初之前就感觉有点不对劲了，只不过自己没太在意。后来，是在女友重新回到身边时，又是激动，又是对自己前一段的生活愧疚，他越来越发现自己出了问题。“近一个月来，勃起总是有障碍、还出现了早泄。”最后，大夫对小王的病情确诊为性功能障碍(ED)。
　　专家指导：纵欲过度容易导致性功能障碍
　　一般来说，勃起功能障碍有两种，一种是身体器官发生了病变，但这种只占到百分之十左右的比例。而另一种功能性勃起障碍占到了大多数，这种病人的身体器官并没有病变，只是身体形成一种特殊病理性神经反射。
　　中医认为肾为生命之本。肾气虚，则由于纵欲过度，造成精气外泄，肾气不足，导致整个人感觉疲惫不堪，精神不振，腰酸腿软，甚至头昏耳鸣，口渴盗汗，还会引发失眠等问题，从而抵抗力减弱，也容易出现勃起功能障碍的现象。
　　年轻人尤其要避免性生活过度
　　年轻人总是仗着自己身体好，年富力强，对性生活就会过度迷恋。在如此频繁的性生活情况下，性系统长期处于一种超负荷工作状态，造成性系统负荷过重，性器官长期充血，没有足够的休息、恢复时间。长期下去，不堪重负的性系统就会出现功能紊乱，勃起功能受到抑制、关闭，就导致了勃起功能障碍的出现。
　　从小王的事情上，专家还提醒青少年不要过早地开始性生活。由于青少年性器官、性心理尚未完全发育成熟，过早性生活，甚至性生活过度，不利于性激素分泌稳定，会加重他们还未发育成熟的性系统的负担，容易最终导致性系统出现各种问题甚至性功能障碍。
　　不要把病症当成简单的性欲过度
　　1.年轻人的性生活，身体强壮的一般以每周不超过2次为正常。
　　2.因为纵欲过度，有勃起功能障碍的男性，可以坚持做运动，同时不要喝酒，不吃辛辣刺激食品。多吃蔬菜水果。还可以食用一些药膳调理。

戒文~念佛戒手淫神法

　　——不屈意志
　　前言
　　念佛降淫念，戒手淫是有方法的，根据本人实践经历感觉念四字圣号“阿弥陀佛”比念六字圣号“南无阿弥陀佛”戒手淫要更好。净空老法师和印光大师都做过开示，虽然念六字圣号和念四字圣号效果一样，但是念四字圣号“阿弥陀佛”更容易摄心，达到专一一念，更能化万念归一念。
　　还有大家不必担心念四字圣号“阿弥陀佛”要比念六字圣号“南无阿弥陀佛”会对佛祖不敬，须知道真正的恭敬是在日常生活中起心动念依教奉行，不犯戒律。十方三世诸佛都希望我们早日往生西方极乐世界不退成佛，这才是佛祖的本愿，如果我们念四字圣号能早日念到功夫成片，一心不乱，蒙西方三圣接引往生西方极乐世界，这才是真正的恭敬！
　　内容
　　当我每次脑海中升起淫念时，或者看到美女时就把美女观想成一具恶臭的白骷髅，恐怖异常，并做不净想，顿时欲火自熄。同时马上严肃起来，观想“阿弥陀佛金色庄严之身相”，心中一句佛号提起，不方便的时候我就在心中默念四字圣号“阿弥陀佛”，方便的时候我就大声大声的念。
　　方法是这样的：（1）淫念一起，首先马上严肃起来，默默观想“阿弥陀佛金色庄严之身相”片刻。（此法中最关键的一步，必须要找一张阿弥陀佛庄严的画像来观想，注意必须是庄严严肃的，阿弥陀佛庄严的的相貌会让你变得坚强而有力的降伏淫念，每次我一观想阿弥陀佛庄严的相貌，心中暖暖的，感觉就象是自己的父亲，心中就升起一种对佛祖的崇敬和向往，此时淫念已熄灭了五分）。
　　（2）观想之后，念之前深吸一口气。
　　（3）吸第一口气，然后边呼气边念（四字圣号“阿弥陀佛”），一口气念5声，吸第二口气也是念5声，第三口气也如此。直到感觉淫念消退为止。（此法中也是最关键的一步，注意必须深吸一口气，而且每吸一口气就观想“阿弥陀佛金色庄严之身相”一次，一口气念完5句圣号，不多也不少。可以用手指数。）
　　（4）念时观想自己念出去的四字圣号“阿弥陀佛”变成一个个大字，流进自己的耳朵里面。达到句句分明，字字入心，发于心，听于耳，入于心，就是注意边念边听了。象这样念特别能够排除杂念，静下心来。
　　（5）念完之后再念一首偈来激励开导自己——“任它淫欲狂似海，紧守海中一明台。声声佛号心中留，妄念欲海自枯竭！！” “明台”的意思就是四字圣号“阿弥陀佛”，就是一句佛号！
　　特别提醒
　　（此法中五个步骤一个也不能少，否则效力减弱，难以达到一招必杀的威力。）
　　念的时候，语速和语调等方法因人而异，不过我建议大家当每次心中淫念或者淫欲升起来的时候试试念念“阿弥陀佛”或者“南无阿弥陀佛”通过多次比较和念诵找出究竟四字佛号还是六字佛号好，前提条件是觉得念着有感觉，喜欢念，能及时熄灭淫念使心中清静，那个方法对于你就是最好的。
　　然后通过一口气念5句或者7句或者8句，找出能及时伏住淫念的念佛号的方法，然后脑海中记下来每次是怎么通过这次成功的念佛号的方法击退淫念，每次淫欲升起来的时候就不要害怕，千万要镇定，回忆起你当时通过念佛成功击退淫念的念佛方法普及到每一次淫魔淫念从心中升起来的时候都按照这个方法去念。
　　念佛戒手淫神法 实践
　　闭目——观想“阿弥陀佛金色庄严之身相”片刻，然后深吸气，跟我一起念：
　　吸第一口气——阿弥陀佛 阿弥陀佛 阿弥陀佛 阿弥陀佛 阿弥陀佛
　　吸第二口气——阿弥陀佛 阿弥陀佛 阿弥陀佛 阿弥陀佛 阿弥陀佛
　　吸第三口气——阿弥陀佛 阿弥陀佛 阿弥陀佛 阿弥陀佛 阿弥陀佛
　　吸第四口气——、、、、、、、、、、、、、、、
　　吸第五口气——、、、、、、、、、、、、、、、
　　念完后再念“任它淫欲狂似海，紧守海中一明台。声声佛号心中留，妄念欲海自枯竭！！”
　　阿弥陀佛
　　这个方法随时随地都可用，(配合其它方法使用效果更好，比如不静观和白骨观。还有念诵经典，培养正气，早上读《太上清静篇》和《心经》都不错。）淫念一起，立马使用，一招必杀淫魔。百试百灵，万试万灵。本人仅此以这些体会和感悟献给各位在苦海中迷惑的同修，希望想戒除手淫的同修们，认真体悟和实践此方法中的五个步骤。四字圣号犹如一把锋利的刀，将把自无始劫以来的淫念一点点化解，甚至彻底铲除淫念，斩杀淫魔，脱离苦海。成功的时后记得不要忘了感谢我，把这个方法告诉别人哦！
　　欢迎流通 功德无量 阿弥陀佛！

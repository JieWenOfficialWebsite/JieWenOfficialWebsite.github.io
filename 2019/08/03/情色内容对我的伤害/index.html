戒文~情色内容对我的伤害

Posted on 2019-08-03

　　所谓“情色内容”是指旨在挑起人的性的欲望和在一定程度上满足这种欲望的一切文字，图片，视频，音频等。英文为pornography。我以前认为消费情色内容不伤害任何人；我现在明白了，伤害最大的是自己。本文介绍我自己对情色的痴迷和我所受到的伤害，意在给后来人以借鉴：前车之覆，后车之鉴。
　　我本人在大学教书（惭愧），有博士学位，而且家人，朋友，同事都认为我是一个好人，善良，又有爱心（这些也都是事实）。但是，鲜为人知的是，我有一个见不得人的嗜好：情色内容。我没有愚蠢到在行为上出轨，但是我有色心。以下介绍消费情色内容给我带来的伤害，包括杀生，对思维和语言的影响，诸事不顺，等等。
　　1. 杀生。我和太太避孕一向都很成功，但是，有两次失败，太太怀孕。我们是不打算要孩子，所以太太坚持要流产。我是学佛的（惭愧之至），坚持要留下来。最终还是流产了。我这两次避孕失败，都与消费情色有关：在这两天的早上，我都背者太太看了情色的内容，晚上欲望就高，结果不知怎么搞的，避孕就失败了。我不是一个糊涂人，太太是医生，避孕还是搞得清楚的，但是那两天，就出差了。一次是我搞错了安全期，太太也没有阻止我；另一次是使用避孕膜，时间没有把握好。我和太太避孕18年，只有这两天出错，而这两天的早上，我都看了情色内容。别人可以认为流产不是杀人，但是我的流产是杀了我自己的孩子。在一次打佛七的时候，我打瞌睡了。朦胧之中，看见在我的身旁，在两排拜垫之间，坐着一个小孩。我猛然惊醒。另外一次梦中，我看见自己过世的母亲，带着一个小孩。这个小孩是我要见的。他没有发育好，四肢扭曲。孩子跟我要一只鸡蛋。母亲的饭锅里清汤清水，只在上面漂浮着一点固体物。我问母亲：“你怎么吃这种东西？！”情绪大动，暮然惊醒。
　　2. 思维和言语。我看情色内容并不常规，所以我可以比较自己正常时和不正常时的不同。我在看了情色之后的几天中，思维变慢，语言不畅。我是讲课的，这个在课堂上最明显：本来知道的东西想不起来了，或者找不到词汇了，或者出现口误了（心里想说甲词，说出来的却是乙词），或者出现口吃了，等等。
　　3. 诸事不顺。我的经历是很顺的，遇事总有人帮忙，但是，在我看情色的期间，却总有不顺的事发生。例如，有几天我在办公室的电脑上看情色小说，那时接到一个电话，说是大学董事会的董事打来的，说是有学生反映我，问了一些班上的事情，问了学生的分数。后来我搞清楚了，原来是一个学生冒充董事，想知道班上其他学生的分数。这类事一般是不发生在我身上的，但是在我不规矩的时候，它就发生。
　　4. 与人相处时不自在。我在看过情色的几天中，在面对别人的时候，不自在，不自然，不自如，似乎心中有鬼，似乎别人都知道我偷偷干了什么。由于心理的因素，语言和表情都不自然。假如这样久了，就会成为性格，别人还以为我就是这样的人。其实不是。
　　诸位朋友，我喜欢我不看情色的日子--我坦坦荡荡，光明磊落，说话（上课）底气也足，语言流畅，思维敏捷，周围的人对我很尊重，我自己感觉很满足。跟看了情色之后的懊恼，愧疚，消沉相比，太不一样了。我本来是清净的。我没有必要过这么脏的日子。
　　最近这半年来我发现念观世音菩萨很管用。《普门品》中说，“若有众生多于淫欲，常念恭敬观世音菩萨，则得离欲。”我每天念1000遍，大约20分钟，欲念明显减轻。虽然还有违犯，但是极少。我的经验是不能等到要去看情色的时候再念观世音菩，到那个时候可能就太晚了，未必能制止我；关键是平时念，每天念，不间断。不能说念了一段时间，看看有效，以为不会犯了，就停下来了。那样不行；一停下来，欲望又升起了，又犯了。
　　诸位朋友，希望您从我的错误和过失当中学到一点什么。这是写本文的原因，也借此机会忏悔我的过失和犯罪。

戒文~戒邪淫在這个世代很重要！

Posted on 2019-08-03
佛说佛法有八万四千种，以不同教法渡世间不同根器的有缘众生。
我的缘分来很突然。回想起来，我是突然对般若心经有兴趣，突然遇到上师，突然皈依了。但修行的过程绝非一帆风顺，现在不敢说有什么大悟，但总算是有点经验，所以希望今天就戒色的问题与大家分享。
因为某种特殊缘分，我承诺过我要起一个戒色论坛，但一直因为杂事太多，所以迟迟未能兑现这个承诺。后来我想与其建立一个新的论坛，可能在已经广泛流传的佛门论坛发表可能得益的人更多，所以最后决定在此分享我对邪淫一事的看法。
我小时候家在香港。在90时代中开始互联网已经开始流行。那时候我刚十岁，也是宿世恶缘深重，我已经互联网的色情世界中泥足深陷，不能自拔。
之后人生旅途中遇到的种种问题，如人际关系，身体状况，心理素质，其实深究之下与沉溺邪淫不无关系。
我自己的苦就不再这里长篇大论的叙述了，但大体来说，苦有以下几种：
1. 身体虚弱，发育不良，一直长不高
2. 精神萎靡，心理不平衡
3. 邪淫永远不能得到满足，所以心中没有安宁。
4. 觉得自己屡战屡败，信心会受到影响
说真的，我也未能将邪淫的习气完全戒掉，因为需要不懈的恒心和耐力。但我发现以下方法可以缓解邪淫的欲望：
1. 淫欲一起，念南无莲花尊丰如来。 这佛与我们特别有缘，诚心念可以马上断除强烈的欲望
2. 但欲望一念接一浪，需要明白为什么要戒色 （如结婚后就以合理的方法和量去行夫妇之礼）， 因为伤身，伤德，影响事业发展。请看：
http://www.jiexieyin.org/ShowArticle.asp?ArticleID=28802
3. 多看一些大德写的书籍，如了凡四训等
4. 如果有佛缘，就透过皈依我佛，透过了解邪淫对身心之害把它彻底的戒掉
5. 因果说得很清楚，如果犯邪淫，你的另一半或女儿就会被人邪淫，或不忠贞。

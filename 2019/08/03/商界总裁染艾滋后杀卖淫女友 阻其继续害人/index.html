戒文~商界总裁染艾滋后杀卖淫女友 阻其继续害人

　　云网10月16日报道
　　近日，红河州金平县勐桥公安边防派出所的官兵们着实紧张了几天。9月23日，勐桥乡一家歌舞厅内，一名男子在包间内将一名女子杀死后自杀，并在自杀前留下遗书。遗书中说，男子与被杀的女子都患有艾滋病，为了不在世界上害更多的人，男子杀死自己心爱的女友后意欲自杀。而遗书中的这句话，成了办案官兵心里的一颗定时炸弹。
　　歌厅发生命案一封遗书吓着人
　　9月23日，勐桥公安边防派出所的官兵们紧张地工作了一天。下午5时30分，边防110值班民警突然报告：辖区某歌舞厅发生凶杀案。
　　派出所官兵很快赶到现场，歌舞厅老板说，一名陌生男子手持凶器挟持一名在歌舞厅工作的女青年，到包间后行凶杀人。为救人性命，现场的官兵用身体撞开反锁的包间门，只见遍地血迹，女青年的喉咙已经被割断，身体被刺三刀，已经没有了呼吸。女青年身旁的男子手腕被割伤，胸部被刺两刀倒在血泊之中。官兵火速将满身是血口中还在呻吟的男子背到两公里外的诊所进行抢救。
　　就在男子被送往医院抢救的同时，还在现场进行清理工作的民警发现，在男子遗物中有两封尚未发出的信，一封是给重庆市某人大的，一封是给公安民警的。当民警查阅信件时，有一行字如同一枚定时炸弹一样：“我叫林凡，男，现年45岁，上海市人;女方，小芳，越南清河县人。我俩都是艾滋病携带者……”在抢救生命时，出警官兵有的手被划破，有的脚被擦伤，好几名官兵身上都沾上了现场伤者的血。
　　9月24日，在勐桥医院的抢救监控室里，受伤的男子从昏迷中醒来，他的第一句话就是：“感谢边防官兵救了我的命。”男子说，虽然他杀了人并准备自杀，但他现在不想死了，因为现在他才知道死亡的可怕。在断断续续的呻吟声中，男子向官兵道出了事情的原委。
　　原本是个总裁内心痛苦动杀机
　　林凡本来是上海商界出名的总裁，因为婚姻破裂而一蹶不振，对林凡爱恨交加的父母在林凡陷入低谷时也双双西去，给林凡留下了一笔遗产。带着父母的遗产，林凡来到重庆打算重新开始，但是，还没开始崛起的他，就再一次被感情重重地打了一拳。就在林凡陶醉在与女友多多的爱情中时，多多却离他而去，并将林凡在重庆的别墅偷偷地卖给了别人。后来，林凡认识了一个朋友，朋友带他到云南，才让他忘却了两次情感失败的痛苦。
　　2004年6月林凡来到云南河口，在河口他认识了小芳，小芳最终成了林凡生活里的第三个女人。2008年，由于两人都没工作，坐吃山空，银行里的存款日渐减少。为了生计，林凡与小芳商量还是找个工作。经过一番努力，两人均被一家外企公司看中。在做上岗前的身体检查时，噩耗传来，林凡与小芳同时被检查为艾滋病携带者。
　　为了治病，林凡卖掉了他俩居住在上海的一幢花园别墅，这也是父母亲留给林凡最后的遗产。但是，卖掉别墅的钱很快就用完了，他们在上海生活不下去了，无奈小芳和林凡于今年6月返回云南河口。因为没有经济来源，小芳只能去卖淫，以此来养活林凡和自己。
　　从表面上看，一切都很平静，但林凡每时每刻都在痛苦地追忆着：是谁害了我?像我们这样的人活在这个世界将会害人，小芳不是在害更多的人吗?一个念头在林凡大脑闪过：“只有死了，一切才会结束。”为此，他首先给重庆市人大写了封长信，恳请人大为他收回送给多多的别墅，还给公安机关写了封信，说明了他杀人的主要原因。
　　目前，勐桥边防官兵身体检查结果已经出来，当天办案的官兵中没有人感染艾滋病，但是现在还在观察中。(文中人名均为化名) 新闻来源：生活新报 (本文来源：云网 )

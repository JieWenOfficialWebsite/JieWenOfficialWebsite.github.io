戒文~当代邪淫果报录3

　　王子提供

　　骗色骗财 搭上一条命
　　某县老大杖子乡村民王某，在河北省打工期间，骗得同厂一女青年好感。王某谎称没有成家，实际上他早已娶妻生子。于2002年9月10日，王某将女青年骗到昌城，安排在亲戚家居住。第二天王某好友梅某来看望王某，梅某知道王某早已成家，不知女青年被骗而来，以为是王某亲戚，竟一见钟情。王某见状，顺水推舟，假说把女青年介绍给梅某，让梅某准备相看钱。梅当回家拿来1000元钱交给王某，并约定9月13日相看。12日上午，梅某一早便来到王某亲戚家，没想看到王某与女青年在亲热，梅某当即向王某索要1000元钱。王某拒不还钱，梅某无奈回家。第二天上午9时，梅某又来要钱，二人话不投机，人财两空的梅某操起木棍打在王某头部，王某当即死亡。梅某也因此进了监狱。

　　都是邪淫惹的祸
　　范子文是北京人，很有才华，但仕途并不顺利，自从学佛后，知道了邪淫的害处，他自己忏悔说：“本来自己并无邪淫的行为，工作中，有异性同事向自己表示过爱意，当时自己未同意，过后也没有行动。但常在睡觉时，在脑海当中想象与那个异性发生性关系的情节，自我认为只是想想，自己绝对不做，没有什么。结果在前几年工作变动中自己莫名其妙地被降级，近来工作也是不顺，面临解聘的危机。为此，范子文幡然悔悟，认为这一切都是邪淫惹的祸，他发露忏悔，愿意写出来，现身说法，以戒来者；他立誓坚决断绝邪淫的意念，做一个清净的人；他恳求苍天给自己一次忏悔改过的机会，使自己能度过这场劫难，能遇难呈祥，逢凶化吉，有一条光明的路。他奉劝仁人志士，千万别涉邪淫，他说，一定要自我约束自己，邪淫是罪业之首啊！”

　　邪淫受人侮辱
　　朱丽华（化名）生得高挑，不胖不瘦，不美，也不丑，属于年轻自然美那种人。上个世纪60年代末高中毕业，被选中做公社广播站播音员。她家在十多里外的大队，朱丽华每天要值机（放有线广播），无法回家。公社为他安排了住宿之处，不久，她就被迫与广播站站长通奸。从内心讲，朱丽华可能不乐意，那时的社会氛围对这种事人人唾骂，尤其是女人，根本无法面对。加上她情窦初开，就这样跟了站长一年多。后来被人举报，县广播站把她关起来反省，当时她二十左右岁，本来胆小，就全部交代了问题。这样一来，本公社就无法呆下去，只好调到另一个公社当话务员。从此臭名远扬，根本找不到对象，仍在单位住宿。因她有前科，公社只要发生案件，派出所的干警就找她羞辱一番，直到三十来岁才与一贫困男子成婚，平时受尽别人白眼和侮辱。

　　邪淫难得亲子
　　1995年候必贵当上了“县太爷”后，认识了杨蓉，一个月后，杨蓉说要找个工作单位，候必贵满口答应，并有了杨蓉。几天后，候必贵把杨蓉安排到粮食局职工学校工作，3个月后，杨又成了县劳动局的工作人员。继而，候必贵又送杨到四川省行政财贸管理干部学院读书，县劳动局将她录用为国家干部，而且还分给杨一套住房。此外，在与候必贵相好的6年间，杨的5位亲友都被县长从即将倒闭的企业调入了县里的要害部门。
　　为了养杨蓉，候便想方设法找钱。他是苦出身，当个县长不容易，不敢大张旗鼓地贪污受贿，他就用赌博的方式“赢” 钱，赢的钱不够，就收礼，1996-2001年春节，先后送杨284万元，他又将杨安置到成都市，买了房子。当候必贵被调查时，才知道杨蓉1999年为他生的儿子，并不是自己的亲骨肉，气得他捶胸顿足地仰天长叹道：“这种打击比被纪委查处还要大”。

　　自己淫秽 丈夫淫人
　　桂清长得并不美，但有一套降伏男人的本事，她在有丈夫的同时，与一副县长有染。副县长对她很关照，住房、工作一手包揽，连她的丈夫也借妻子的光。桂清与副县长的关系在县城是公开的秘密，所以，有人想到副县长处走后门，找桂清即可，桂清与情夫双入双出，又对丈夫严加看管。其夫在武装部工作，一次民兵训练时，与一女民兵合一张影，被桂清看见，不问青红皂白，找到那女民兵，打折了鼻梁，被起诉犯了伤害罪，幸副县长多方周旋，桂清才保住公职。桂清虽严管丈夫，丈夫也不甘心独守，在单位与一女同事秘密来往，已经多年，等桂清知道，全县机关干部早已经知道多时，为此，桂清指责丈夫，丈夫指责她。各自都有自己的情人，最后只好谁也不管谁。各自找各自的情，各人有各人的生活，貌合神离。

　　淫秽伤人猝死
　　刘王点（化名），是某县副县长。他四十出头，风华正茂，上边有人撑腰，在县里亲戚、朋友、同学盘根错节，势力熏天，但某天却猝死，才四十三岁。
　　说起刘王点，他最爱做的事是找女人，似乎一天也离不开。一次，他找来“三陪女”做爱，不知为什么，咬掉了小姐的奶头，被小姐告发。为小姐治伤、赔偿损失，他花了几十万元。当然，钱是贪污、受贿、收礼来的，靠工资，他挣不来几十万元。此事闹得沸沸扬扬，为保住官职，刘王点上下打点，又花了上百万元，才保上下平安。过了两、三年，舆论刚平息，人就死了。

　　风流并不快活
　　吴超凡（化名）是某市市委书记，由于有权，用公款养了数个女人。这些女子个个年轻漂亮，即有文化，又娇柔可人。市委书记是一个市的土皇帝，他操人事生杀大权，翻手为云，覆手为雨，真是前呼后拥，风流快活，呼风得风，唤雨得雨。谁知，转眼7、8年过去了，吴超凡到了退居人大的年令。为保持余下的势力，吴超凡谋取市人大主任的位置，并按省里的部署初步组阁市委、市政府班子。因吴超凡各种错误太多，在他将失去市委书记位置的时候，百姓的上告信如雪片般飞往省委。省委决定把他调走以平民愤，于是立即任命了新的市委书记，吴超凡调省人大任副秘书长，吴超凡终于在仕途争斗中落下了马。逢年过节，某市的市有时会给寄一张明信片，写着吴超凡大贪官收。他的女人们，一个也不能跟去，平时在单位仗着市委书记的势力，横行霸道，吴一走，她们立即失去了势力，都很不得志。

　　才子被邪淫拉下马
　　提起刘贤民（化名），熟悉他的人都说他有能力。他是文革前的名牌大学生，文革中虽是“臭老九”，没有爱到什么冲击。十一届三中全会后，渐渐重用知识分子，一些大学生从工厂、学校提拔为县级领导，常常为找不到真正的大学生发愁。而有创新精神，有发展思路名牌大学毕业的刘贤民一直没得提拔。
　　刘贤民虽生得黑，却是一表人才，相貌堂堂，人称“风流才子”。刘贤民就是有这两下子，一个再乱的单位，他去几个月就会有很大的起色，但他走到哪里，最后都因男女通奸问题臭了一处。一年，他在全县唯一的重点校--一高中任校长，因与一个女人通奸，激起教师们的愤怒，以至于把他围住，不准离校，向他讨要说法。县委常务副书记与他私交很好，书记夫人因为为人正气在全县爱戴。为给刘贤民解围，夫人出面劝说，并亲自用手拉着他，保护着刘贤民走出学校。
　　事后，刘贤民受到处分，被降职安排，后来一直教书，十几年下来，刘贤民可谓桃李满天下，但他始终是一个正科级干部，没有提起来。

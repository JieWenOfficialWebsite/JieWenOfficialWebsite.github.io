戒文~恶毒的贪欲使人全身腐烂

人死之后为什么会全身腐烂????(图片:http://www.fosss.org/wczp/fulan.JPG)就是因为人有恶毒的贪欲,(还有嗔恨等)所以常劝世人修不净观，观想尸体腐烂的景象，那些全是因为恶毒的贪欲造成的。为什么那些大师死后身体可以不坏？或者有舍利子？除了他的功德以外，还有一个原因是他息灭了那些恶毒的贪欲。修不净观的人要明白这个道理。恶毒的贪欲是使此身腐烂的主要和根本原因。所以修不净观可以去恶欲，转化此色身。当然不是要你贪恋此身，因为恶欲不但腐化你的身体，更恐怖的是腐化你的灵魂．以下是宣化上人的开示：

佛说：「一失人身，万劫不复。」....... （ 所以一个人投生去做畜生，不是一个人做一种畜生。他会分开：一个人做马、牛、羊、鸡、犬、豕，这是身体比较大的，有灵性聪明一点的畜生。这一个人又可以变成七种的畜生，因为经过分开了，这叫「分灵」，便性化灵残。

他的灵性既然分开了，又往更小的再分。譬如马变鸡鸭，又变成很多了。一只马，又可以变成七只鸭、七只鸡等等。既然又分开了，那点灵性又少一点，又愚痴一点。

因为这种关系，所以佛说：「一失人身，万劫不复。」若把人身失掉了，百千万劫再也不能恢复。为什么呢？因为他已被分开了，经过化学工厂的化验，分开一部份一部份的。假如人会变成蚊虫这就是因为人的心很毒，专门贪而无厌，总想吸食人的血。这种思想长久了，就会变成蚊虫，而遂心满愿的到处去吸人的血。牠虽然是性化了，可是牠还是有个灵性，还有点知觉性，于是牠也只知道吸人的血，旁的什么也不知道了。牠之有生命，也就是以贪心来为牠的工作，这是蚊虫的因缘。

蚂蚁呢？牠们都很好吃的。一个人要变蚂蚁，可以变成八万四千那么多。所以你看蚊子，一班一班总有好多在一起。蚂蚁一窝一窝，也是很多很多聚再一起。牠们因为性化灵残，所以其力量也小了，没有那么大的力量。 ；）就是说人死之后如果落入三恶道，灵性将会分裂．．．．．

　　以下是宣化上人的开示：（人身上的蛆虫：有了贪心、瞋心、痴心，就生出贪欲、瞋恚、愚痴的虫子。

《华严经》上说：在不可说诸佛报身上，各各现出不可说诸毛孔。在每根毛孔中，又现出诸佛剎土，又现出大宝莲华，又现出不可说的莲华叶。在每片叶中，皆放光明。在光明之中，又放光明，现出种种色相，无穷无尽。在光中现月，月中复现月，这种境界，都是诸佛所现。

我们众生的身上，也有所现。现什么？现出虫子。在我们身上有无量的虫子，这虫子比厕所的蛆还要臭。在每条虫子的身上，又现出不可说的细菌。这种细菌，凡夫的肉眼是看不见的，必须是证得天眼通的人，才能看得见 ( 现在特大号显微镜也看不见 ) 。

这种细菌，非常肮脏。不过，我们自己不知道而已。把大虫子 ( 身体 ) 看得那样的宝贵，那样的重要，又怕饿着，又怕冻着，又怕热着，想尽办法来保护这条大虫子。为什么这样的爱它？因为有三毒之心。有了贪心，就生出贪欲的虫子。有了瞋心，就生出瞋恚的虫子。有了痴心，就生出愚痴的虫子。总而言之，有了什么思想，就生什么虫子。

我们的思想，比厕所的蛆虫还要肮脏十倍，尽想些邪知邪见，没有一点正知正见。思想越是肮脏，越想要肮脏的东西，乃至造成肮脏的世界，染污的世界，所以变成五浊恶世。五浊恶世，是由我们不清净的妄想心所造成的。这个娑婆世界，并没有浊，而是我们把世界弄浊了。五浊就是劫浊、见浊、烦恼浊、众生浊、命浊。这五种浊，多么可怕呀！

我们心中有不正当的念头，例如嫉妒的念头和障碍的念头，时时在作怪，可是自己还不承认，反而把它覆藏起来，以为人不知道。其实这种行为，比妖魔鬼怪还要丑陋十分。丑陋的程度，简直是等于放出臭气，熏到三十三天去，把玉皇大帝熏得掩鼻而逃。你们看！这股臭气厉害不厉害？可知其臭不可闻也。

自己的思想和行为，就是这样的臭，自己还不知道，认为自己是清高的人，那知把世界都给染污了。因此，变成五浊恶世。各位要知道，我们一念为正，就能补助天地之正气；我们一念是邪，就是臭气冲天，令天地容纳不了。这不是讲笑话，更不是戏论，的的确确是这样子。为什么天旱不雨？就因为人所放的臭气，冲到虚空中，把兴雨的龙，给熏跑了，无法执行牠的任务。））

　　　淫欲重的应该常修此不净观，莫让恶欲腐化了此身和灵性．

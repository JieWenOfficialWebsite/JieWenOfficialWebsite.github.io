戒文~大四男生带着奶奶上大学

Posted on 2019-08-03

孙宇和奶奶

本报吉林市电 一个只有15个月大的孩子在父母离异后，被丢给了年近六旬的奶奶。二十几年来，祖孙俩一直过着近似漂泊的日子。这个婴儿就是孙宇，目前是北华大学师范外语学院的一名大四学生。令人难以想象的是，为了可以继续读书，初三毕业后，孙宇就开始想方设法挣钱养家糊口，卖雪糕、烤苞米、当家教……经过努力，他不但为自己挣到了上高中和大学的学费，还挣出了和奶奶的生活费。为了更好地照顾奶奶，大二以后，孙宇将年过八旬的奶奶接到身边，虽然祖孙俩租住在一个面积不过十几平方米的平房里，可对孙宇来说，这就是家……

童年和奶奶一起漂泊

回忆起童年的生活，孙宇形容自己和奶奶就像一大一小两个浮萍。奶奶告诉他，在他只有15个月大时，父母离了婚，自己从此被丢给了奶奶，父母很少来看他。在他12岁时，父亲又因肝硬化去世。此后，他很少能见到母亲。孙宇说，他如今最大的缺憾就是没有体验过来自父母的爱。


奶奶刘桂琴没有收入，他和奶奶也因此居无定所，经常寄居在几名亲戚家。孙宇称自己小时候非常自卑，就像一棵没根的草。可骨子里，他又有一种不服输的劲。他没资格与同龄人比吃比穿，惟一与同龄人可以比的就是学习成绩。因此，从小学到大学，他的学习成绩从来没下过前两名，且一直担任班长。


挣出学费和奶奶的生活费


因为很早就听说上高中花费很高，他担心因此被迫辍学。于是，初三刚毕业，他就盘算着该如何挣钱了？与另外两名同学商量后，3人决定卖雪糕。他们用纸壳自制了一个雪糕箱，随后推着自行车去吉林市桃源路一家冷饮厂花18元上了一桶雪糕。走街串巷卖雪糕的第一步就是吆喝，可当时他和另外两名同学都张不开嘴。三人只好用石头、剪子、布的游戏来决定谁最先吆喝。当第一次大声喊出：“卖雪糕”时，他感觉自己的脸特别红。将这桶雪糕卖掉后，两个同学放弃了这份工作，而他则沿街叫卖了一个假期。有时在午夜11点，某些居民楼附近仍能听到他的叫卖声。


从此，每个假期他都努力挣钱。由于卖雪糕利润较薄，他又改行烤苞米。每天天一放亮，他就骑车去珲春街上苞米，回到叔叔家后则赶紧支起烤炉。只几天，他的双手就被炉火熏烤得裂了口子，很痛。满是汗水的脸也变得黑黑的，可孙宇觉得值。因为挣到钱后，就不用奶奶再为自己的学费发愁了，自己还可以买想要的参考书。最重要的，还可以为奶奶买一些老人家从来没有吃过的零食。


2002年，20岁的孙宇以504分的成绩考入北华大学师范外语学院。当时，奶奶为每年4600元的学费愁得吃不下饭，甚至打算让他不要再念了。可孙宇对奶奶说，他不相信靠自己的努力圆不了大学梦。就在这时，命运给了孙宇一个意外的惊喜，他获得了国家奖学金。不但拿到4000元钱，还免了一年学费。孙宇告诉记者，通过卖雪糕、当家教，如今他不但没欠学院一分钱学费，还挣出了和奶奶的生活费。


尽一切努力让奶奶舒心


与奶奶相依为命20多年，孙宇印象最深的一件事是：有一年，奶奶领着他去医院看哮喘病途中，他被街边卖的元宵吸引，奶奶于是给他买了半斤。可看病回家途中，奶奶不小心摔了一跤，头部着地的奶奶当时就昏了过去。回到家后，受伤的奶奶没顾上休息，却直接进厨房给他煮了元宵。因为太馋了，半斤元宵几乎都被自己吃光。想起当时舍不得吃一口的奶奶在一边幸福地看着他，如今他十分后悔——当时怎么就不知道让让奶奶呢！


如今，自己已经长大了，孙宇认为照顾奶奶是自己的责任。大二时，他和奶奶在学院旁租了一个十几平方米的平房，这也是他第一次有了属于自己和奶奶的家。孙宇说，他最开心的就是看看奶奶不经意间掠过的幸福感。一次，他买了一个火龙果。虽然自己从来没有吃过，可看着奶奶细嚼慢咽的样子，他觉得特别欣慰。他曾在奶奶过生日时送了她一双鞋。看着奶奶穿起这双鞋走路十分小心的样子，他说自己特别想哭。一个月前，已83岁高龄的奶奶不慎摔了一跤。由于伤及韧带，奶奶如今大多数时间只能躺在床上。孙宇说，看着奶奶痛苦的样子，自己的心就像被捅了一刀。奶奶为自己辛劳一生，自己什么时候才能让她享受到快乐的晚年呢！


孙子有出息 奶奶笑得很灿烂


孙宇告诉记者，多年的奔波让他感觉有些累，可他觉得自己有一个责任，并由此生出一分自信。孙宇说，以前自己很少笑，可现在自己脸上的笑容却越来越多。记者了解到，孙宇目前已与辽宁省一所学校签了用工协议。孙宇说，自己将这个好消息告诉奶奶时，奶奶笑得特别灿烂，她再也不用为孙子到处挣钱提心吊胆了。孙宇说，他前几天已与奶奶商量好，将来带着她到辽宁一起工作。可这几天奶奶又变卦了，因为怕自己拖累孙子。孙宇说，他会继续说服奶奶，因为奶奶是他永远的牵挂。


耳朵有点背的奶奶刘桂琴对记者说，因为有了个孝顺、懂事的孙子，自己很知足。虽然为孙子吃了不少苦，可现在觉得只要孙子有出息，自己再苦再累也值，没有什么比看着孙子有出息更让人欣慰的了。


他教会我们从不抱怨


记者到北华大学宿舍采访了孙宇的部分同学。“你很少看到他静止的时候。”一名叫崔宏宇的学生告诉记者。在他眼里，孙宇似乎有一种取之不尽的力量，而这种品质很难在其他学生身上发现。一名叫汪泽的学生告诉记者，他从孙宇身上看到的是坚韧，孙宇身上那种永不服输、从不抱怨的乐观精神对他的影响特别大。还有同学向记者介绍孙宇的另外一面，那就是他还是一个多才多艺的班长。记者了解到，喜欢唱歌的孙宇被很多女同学称为“小刘德华”。

孙宇的老师吴常铭向记者介绍，孙宇有着与同龄人不一样的人生经历，因此也显得比同龄人更成熟，在责任感方面显得尤其突出。因此，他很快被学院发展为预备党员。她相信，一个有责任感的人将来一定会有所作为。另外一名王老师告诉记者，学院的贫困学生很多，但完全实现自给自足的却很少。为了弘扬孙宇这种刻苦求实的精神，学院曾安排他给刚入学的大学生作报告，为的就是让新生尤其是家庭贫困的学生从他的身上学到自立自强。(记者 李静 报道/摄 )

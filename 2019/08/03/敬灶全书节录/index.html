东厨司命通天定福奏善真君劝孝文

孝之心，人人可以尽；孝之道，人人不能尽。试问身从何来？亲为生我之人。孝为何事？人所固有之心。人当少时，在乳哺孩提之际，未有不念父母者。至十岁外，知识渐通，性情渐变，恃父母爱怜之余，使唤之未必顺从，呵责之反生怨恨。不孝之迹，日已形矣。十六以后，情窦乍开，倘不以圣训格言，时时讽诵，警惕其心，或误交狎邪之友，遂致迷恋烟花，伤身废业，是父母痛心。不孝之罪，何可赎乎？迨娶妻生子，琴瑟恩深，椿萱爱薄。依房内之娇妻，顿以双亲为外客；听枕边之谗语，甚以二老为仇人。于是耽酒色而致成疾病，父母虽有言勿听也；有财物而私其妻孥，父母虽欲用不能也。贫贱之子弟，怠情习邪，不自劳而劳亲；富贵之子弟，骄奢作孽，每自害以害亲。妻子有疾病，忧之虑之，必求治而愈之，疾病在父母，不若是之急也。妻子之衣服，养之给之，必多方以备之，衣食在父母，不如是之切也。至如父母，或有爱怜少子之心，及父有续娶及买妾生子者，人子之居心更不可问矣。呜呼！父母之恩，昊天罔极。乃不知有以报答，反多落寞心、猜忌心、刚愎心、愤恨心、贪淫心、嫉妒心，种种非心，不孝之罪，上通于天。犹欲妄想读书成名，生理发财，家庆团圆，嗣多贤哲，何可得乎？余为司命，日在人家，记功记过，历历不爽。至每月晦日，合家家而汇奏于天。无如尽孝者甚少，不孝者甚多。于是按善恶为报应。天即有好生之心，人实有自召之劫。余深悯焉！爰降策于乐养居，特劝世人。知孝道者，益尽其心。其素来不孝者，即速速猛省回头，承欢膝下。身后之丰祭皆虚，急向生前奉养；庙内之烧香尚隔，须从堂上瞻依。勿尚虚文，而将实意；勿推同气，而竭一身。庶几一人孝，人人孝；一家孝，家家孝。孝为顺德，和顺之气上充，而何有于逆境乎？并请减省一切日用、起居、饮食、衣服，浮靡无益之费。推尽孝之心，以力行善事。自见厨灶安然，门庭无恙。化恶星于四季，迎吉曜于三宵。而何必效纷纷趋避之私智哉？

劝孝文多矣！皆言父母恩德，以动其良心。此篇极言不孝情迹，以冀发其羞愧悔悟之心，是逆流而挽之法，当头棒喝。若读之而不回头者，非人也！

东宫司命通天定福奏善真君劝善文

吾乃东厨司命，受一家香火，保一家康泰，察一家善恶，奏一家功过。世上之人，如在梦中。光阴虚掷，只图快乐。不修功德，不信三教，不惜字纸，不敬五谷，不孝父母，不慎言语，不忌暗室，不净心地，不悔前非。知过不改，知善不为。甚可惜也！夫善善恶恶，据三尸神所奏，大小存簿。每逢庚申日，上奏 玉帝。善恶簿呈殿，终月则算。功多者，三年之后，天必降之福寿；过多者，三年之后，天必降之灾殃。如有曾犯过恶，后自改悔，一旦回头，神钦鬼伏。斯言虽小，祸福攸关。慎之！

灶王真经上卷

志心皈命礼

身原甲乙。命数东宫。惟始炎德以化身。尊秉五星而首用。烧丹炼汞。尅西生中。或布云而行雨。或秉炁以化凶。无情有事。填实白空。家家户户。

司命之尊。大悲大愿。大圣大慈。东厨司命。灶王府君。恭炷真香。虔诚上启。

太上无极大道。三十六部尊经。 玄中大法尊师。

道言昔登昆仑之山。有一老母。上通 天界。下统五行。达乎神明。关乎二炁。在天则为 天帝。在人乃为 司命。又为北斗 七元使者。主人寿命长短。富贵贫贱。管人职禄。掌人住宅。十二时辰。每月朔日。记人善恶。录其轻重。夜半奏上 天曹。定其簿书。悉是此母也。敢问斯言。幸垂教示。是以合众竭诚。皈神皈命。人各恭敬。 天地临位。 司命灶君。世人恭敬。福禄咸宁。去除灾难。疫疠潜奔。仰依诲示。普得安宁。三昧火出。五色烟生。人不知其中妙趣。故不免一顾神惊。以土为安宅。视水若比邻。无论富贵贫贱。皆与吾相爱相亲。也不须朝山拜佛。也不必礼忏讽经。只要你把吾前打扫得洁洁净净。本是吃斋体。难将五荤禁。善善恶恶在吾心。每到庚申月晦夜。虔诚敬茶一瓶。信香三炷。人把我早夕恭敬。我包彼福寿康宁。

灶王真经中卷

天地开辟。吃的是毛。喝的是血。爰钻木而取火。百姓才知烹饪。佐太一造化之功。助浑提元孕之妙。是为 太上无尊之神。管人善恶。恒河沙数民。恒河沙数神。恒河沙数身。世上无处无我。敬之以香灯。不知敬我以洁净。孝你父母。才是敬我。和你兄弟。才是敬我。妯娌相睦。才是敬我。夫妇如宾。才是敬我。能信神明。才是敬我。厨登牛犬。秽气触我。我必降灾。妇女不洁。臭气冲我。我必降灾。一家吵闹。使我不安。我必降灾。行诸恶事。视我如无。我必降灾。每月晦日。奏人善恶。 上帝耳目。实我是岢。 上帝喉舌。实我是司。吾神不大。吾神非小。自有天地。即有吾神。祸福之权。 上帝授我。我是其人。降以祸福。士不入我禄籍。我不奏 帝。难望功名。农不入我福籍。我不奏 帝。难望发达。工贾医卜。我不佑彼。无处可行。凡是人类。以我为主。大恶大灾。小恶小灾。 上帝赐福。先问于我。 上帝降祸。先问于我。 天上无我。耳目无人。地下无我。生死不分。家中无我。善恶不明。风雷听我号令。雨旱由我传奏。世上之人。速速敬我。各人想想。不敬父母。天雷劈之。不敬尊长。王法处之。不敬三宝。刑祸随之。不敬吾神。即不敬 天。不敬吾神。即不敬地。不敬 天地。 天地不容。不敬吾神。吾能容乎。降以灾病。暗夺冥纪。家内小故。皆吾所使。家内小祥。皆我所佑。每日晨起盥洗。焚香恭读一遍。心体力行。求子得子。求富得富。求寿得寿。妇人念此。百病不生。家长念此。百福骈臻。有事念此。转祸为祥。无事念此。添子益孙。富者念此。永远不贫。贫者念此。得利致富。若谤吾言。吾神削纪。吾神至尊。其勿忽吾。

灶王尊经下卷

志心皈命礼

命承北斗。位镇东厨。为五祀之尊。得九灵之道。作 上天之耳目。司上下之灾祥。记录功过。权衡善恶。造化悉由于水火。万民皆赖为陶甄。祸福总归于善恶。六耗悉皈于醮谢。四时迪吉。八节呈祥。叩之则应。有感必通。

大悲大愿。大圣大慈。本家司命。太乙奏善。灶王府君。

吾以道果圆满。得镇斯位。作人间司命之主。为 上天耳目之神。上自天子。下及庶黎。咸皈掌握之中。群入注录之内。因有善悉陈于 天。凡有恶总申于地。其间男女。有不忠不孝。不仁不义。不友不恭。不良不顺。以致大奸大恶。有悖纲常。有乖伦理。一一记载。无俟咎刻。即以升闻。或当富而夺其籍。或当贵而除其名。或当长寿而易为短命。或当安存而转作危亡。依人所犯。以加谴责。世人不悟。自谓己所独觉。人不知见。殊不知暗室亏心。神目如电。意念起时。早已鉴之。彼人身中。有三尸神。有本命星官。本宿神将。又况家有先灵祖考。户有纠察神君。兼之香火福神。无鞅圣众。布列森严。为善为恶。鬼神断无不知。吾之职司一家之性命。未尝渗泄分毫。不曾姑宽寸尺。善即表扬。恶即震怒。视四海之人心。如粟粒窍。操六合之功过。犹如反掌。无论贵如帝王。显如卿相。贱入编氓。愚如悍妇。悉不能外。一日有十二时。时时俱在。一年有十二月。月月不遗。古初人情淳朴。无诈无虚。不欺不诳。 天道佑之。鬼神护之。时和年丰。民安物阜。沴气不行。一切灾害妖孽。俱不得肆。上兆景星庆云之祥。下征风调雨顺之瑞。君民安享其业。斯称盛世。近日人心奸宄。子不孝于亲。臣不忠于君。卑幼不敬于尊长。媳妇无礼于舅姑。甚而动口咒诅。指天誓地。每日只以使心用心为能。终岁但以瞒人骗人为智。愚痴蠢妇。不柔其性。在内强悍高声。或对灶而妄言绮语。或对灶而歌吟哭泣。口中不洁不净。骂儿骂女。怨天怨人。阴触鬼神。莫此为甚。有种饕餮。徒贪口腹。不念牲命。宰牛马。烹猪羊。割鸡犬。杀鹅鸭。以及飞潜等物。罗而登之于庖厨。冤气不息。忍于见闻。只求适口。妄为造孽。如此之人。罪不容逭。故不惮殷勤。详为开谕。富贵之家。吾司其命。贫贱之室。吾宰其权。即以巢穴以居。蓬茅而处。凡炊爨所在。吾即在焉。显化一身。不见有余。分而百千身。万亿身。恒河沙数身。不见不足。灵通三界。神遍九天。正直不曲。公平无私。所喜者家有孝子。可爱者国有忠臣。犹冀兄友弟恭。夫和妇顺。广行善事。多积阴功。知过即改。知善即为。弥天之罪。一悔便消。百全之行。即时当勉。若人于每月三十日。读诵此经。可以削其怨尤。若斋醮之家。能诵此经。当为保奏。善功完结。即降吉祥。学道之士。切勿勤始怠终。有误前程。居室之人。亦须革故鼎新。以资源本。无贵无贱。共宜兢惕。遂前世之因。不堕于今生。兹今生之因。克培于后世。八难三途。苦趣永免。四生六道。恶报长消。九祖先灵。同登极乐。五亲白丁。共入莲台。乃说偈曰。

太上灶神。奏善府君。烟火在处。即写真形。

视之不见。听之不闻。善善恶恶。毫厘分明。

有善必注。荐扬帝庭。以彰善报。子贵孙荣。

有恶必录。上呈玉京。以示恶报。身灭家倾。

每当月晦。一度一呈。天地覆载。大德日生。

广运无边。至公至仁。世能体此。永享太平。

说偈已毕。声彻上下。功参乾坤。

而时。神将猛吏。童子郎君。咸稽首礼谢。踊跃欢喜。信受奉行。

收经赞

监斋大神。司命灶君。永镇家宅保安宁。善事奏 天庭。恶与无闻。盛德合乾坤。

太乙奏善灶王府君三称

愿此诵经功德。祈福赦罪消愆。一门老幼吉庆。

四时灾疫远遣。有求悉蒙感应。身谢保奏升仙。

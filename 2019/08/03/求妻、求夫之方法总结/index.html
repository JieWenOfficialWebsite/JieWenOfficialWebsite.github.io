戒文~求妻、求夫之方法总结

　　求妻、求夫之方法总结
　　文章转载
　　末学最近一直在潜心研究关于妻求夫求法侣的方法，我请教了身边修行好的师兄，看了许多佛教经典书籍，浏览了一些大德开示，听佛教讲座，此讲座讲的是一位美国籍藏传佛教格西学位成就者麦克格西关于通过运用金刚经的智慧内涵求法侣的开示的专题讲座，在这里把讲座精华的内容介绍给大家。
　　在我的方法总结中，主要分为八个大的方面，分别是：忏悔，戒律，修行，慈悲，智慧，德行，福田，信心。这八个方面是同步进行，密切联系，不可分割的，同等重要的。
　　印光大师法语 ---念观世音菩萨求妻得妻，求子得子
　　人生世间，父母，寿命，相貌，学问，夫妻，儿女，皆是前生所作之业之所感召。若有大功德，则会过于前生所培。若有大罪过， 则便不及前生所培。是以要认真修持，以转前业也。汝欲得有学问，有才能，有德气之好丈夫，或恐汝前生未培到这个福，则便不能满汝痴心。楞严经谓，念观世音 菩萨者，求妻得妻，求子得子。求妻于菩萨，谓求菩萨加被，得其贤慧福德之妻。汝求夫亦然。极力念观世音菩萨，当能满汝所愿。否则，纵得好夫，或者又有短 命，多病，祸患等事，况未必能得好夫乎。汝切不可以貌美起骄傲心。此心不息，便难载福。汝能谛听我语，则汝之所受用者，皆有超过前生所培多多矣。（与周福渊女士书）
　　【1】忏悔：学佛最基础的是要信因果：欲知前世因，今生受者是，欲知后世果，今生作者是
　　我们为什么会遭遇各种不幸，就是因为无量劫以来伤害过别人。因此我们应该升起猛烈的忏悔之心，诚心改过，通过修行和自己的实际行动诚心改过。发愿纵遇命难，永不造罪！
　　我强烈建议：修金刚萨埵心咒、金刚经。不论是显密，不论是哪位大德，都是非常推崇金刚萨埵心咒。
　　大家都知道，金刚萨埵心咒在忏悔的法门是最殊胜的，感应最强的，五无间罪都可以忏悔的，只要你精进修行。具足信心观想金刚萨埵佛加持自己，并具足信心去念心咒或者全咒百字明，也是非常殊胜的。念咒最主要的是信心。
　　忏悔自己生生世世所有的罪业，功德回向法界一切的众生解脱成佛，回向给自己累世的冤亲债主早日解脱！
　　【2】戒律：大成就者法王如意宝说过：清净的戒律是佛法的基础！
　　为什么这么强调戒律呢？首先持戒有功德，持戒持的好就有护法保护，为我们修行和生活遣除违缘和障碍。尤其是杀生戒、邪淫戒、饮酒戒一定要持好。
　　杀生，是罪大恶极！
　　邪淫，是导致婚姻不顺、事业不顺的头大号关键因素。我身边的很多爱看黄书、爱手淫、爱婚前性行为、多交滥交的师兄们，他们的婚姻和工作很不如意，这一点我可以发誓我所说的是真的，绝对的因果！
　　不看黄书、不手淫、不婚前性生活、不滥交，这是最起码要做到的。
　　还有一条很重要的是：不破坏别人的家庭，不与异性搞暧昧。
　　其次，饮酒戒也很重要。一旦饮酒、酗酒，很容易导致邪淫等其他戒律的坏失。世尊曾说：喝一滴酒不是我的弟子！
　　假如你没有受戒，建议你去寺院受戒，因为持戒是有功德的，假如你没有受戒又觉得自己现在没有能力持戒，那先不要受，自己在生活中先学着持五戒，努力去做，有一点能力的时候再去受戒！
　　【3】修行：(本人认为最快的还是念观世音菩萨求姻缘)
　　下面我要讲在求伴侣的法门中，哪些是比较快的。这里我推荐三种：1：地藏经 2：金刚经 3：作明佛母心咒！
　　1：地藏经，地藏菩萨
　　世尊曾经说过，要想求愿望，地藏经菩萨是感应最强的，是满愿王。有能力摆一个佛堂每天供养礼拜地藏菩萨更好，地藏经中也讲过求愿的问题，只要一心瞻礼，供养地藏菩萨，每天念地藏经，地藏菩萨圣号，多做善事，无愿不满！
　　我的建议，在地藏菩萨前，发愿读多少遍《地藏经》求一个和自己一起修行一起做善事，生活中相互扶持的清净如意伴侣，但是这个发愿最少是一百部《地藏经》，同时每天持地藏菩萨圣号1000遍。一定要记住每天不能间断，如果实在不得已，要忏悔，念诵地藏菩萨圣号5000.每天做的功德回向冤亲债主，祈祷地藏菩萨加持他们早日解脱往生极乐世界。
　　念《地藏经》超度冤亲债主是最合适的。
　　2：金刚经的方法：
　　下面我带来一位美国籍藏传佛教格西学位成就者麦克格西在弘法利生的事业中关于通过运用金刚经的智慧内涵求法侣的开示！
　　伴侣从哪里来？
　　答：从心里来！
　　金刚经中云：菩萨若有我相，人相，众生相，寿者相，即非菩萨
　　在这个讲座中指出，一切法从心想生，你的伴侣是从你的心里来的，你看到你伴侣的缺点，这说明你也有这个缺点，你希望你的伴侣具备什么样的优点，那么你要首先自己做到，假如你发现你的伴侣有什么缺点，你不需要和他吵，不需要和他谈，你需要先改正你的缺点，然后自己为他树立榜样，这样你渐渐的会感化他，他自然会佩服你，相信你，然后改正！你有什么样的德行就会感召什么样德行的人。所以说物以类聚，人以群分！
　　在这个讲座中还指出，怎样遇到伴侣呢?
　　你不需要刻意去找，更不比烦恼上火，你需要做的是两点：
　　{1}：准备好你自己：
　　伴侣从自己的心里来，在你遇到你的伴侣前，你首先准备好自己有一个好的德行，只有好的德行才能影响别人，感化别人，才能通过自己德行的力量感召一个同样德行好的伴侣。德行主要指人格健全，真诚善良，忍辱奉献，不看别人的缺点，多反省改正自己！
　　{2}：你最主要的任务就是修行！
　　怎样修呢？发愿读金刚经！每天坚持不间断，金刚消业障非常快，可以遣除我们的违缘，同时开启我们的智慧，金刚经开智慧很不可思议，通过开智慧让自己树立正确的观念，放下执着烦恼，正确的处理问题和看待问题！回向和上面一样的。
　　3：下面隆重介绍作明佛母心咒的方法：
　　因为作明佛母是密法，修法的话需要传承，但是作明佛母心咒是不需要传承就可以念的，具足信心加持力非常大。作明佛母是观世音菩萨的一个化现，是非常厉害的怀爱本尊。她的主要作用是：
　　可增长无量福报功德，财富增长，学佛修行之资粮无缺，增长人缘、权势，受部属、亲友爱戴，谈婚论嫁，能圆满各种世间与出世间之事业，凡一切有情应感恩此佛母之大悲种性。
　　如果你想要修持作明佛母心咒，那建议发愿念100万心咒：嗡 咕噜 古咧 梭哈
　　念的时候，观想自己就是红的的作明佛母，观想她的样子，法器，然后专注念，渐渐升起猛烈的信心，相信佛母的加持一刻也没有间断！
　　光念咒还是不行的，因为我们凡夫业障深重，心思散乱，在你修作明佛母心咒的同时，兼修地藏经或金刚经。这两本经，你可以选一个，配合作明佛母心咒一起修！
　　回向同上面一样，再加一个希望佛菩萨加持我遇到一个一起修行的清净如意伴侣！
　　【4，5】慈悲与智慧；
　　根据地藏经中云，想要得到菩萨的满愿，一定要具备以下几个条件！
　　1：善男子，善女人！是指的发大乘菩提心的善男子善女人，就是我们心内所想的，行动所做的，都是有利于利益众生的，首先想到众生的！
　　2：发阿耨多罗三藐三菩提心！这个意思是说我们要放下世法，投入到精进修行成就解脱利益众生的道路中去，我们的愿望可以求，但是不执著，并且我们的愿望是善愿，我们的愿望要有利于修行
　　3：这是对慈悲的要求，对智慧的要求呢，就是读金刚经，开启自己的智慧，放下智慧与烦恼！
　　生活中多去帮助众生，你只有多去帮助别人，才会得到别人的帮助，不论是伤害你的人，还是帮助你的人，只要她需要帮助，我就去帮助他
　　有位大德说：只要你发出了真心想修行的心，并认真去做了，只要你力所能及的去利益众生了，不计较利益的利益众生了，那你世间的问题就不用担心了。
　　【6】德行：
　　修持自己的德行：厚德载物，人若是具备了一定的修养与德行，会感召很多善缘。首先要学会忍辱，把心量放大，其次要常常微笑，乐于助人，乐善好施，尽量做事情多为他人着想，把自己的利益看得轻些。你只有把你的德行提升了，才能感化别人！
　　心胸宽广，忍辱负重，奉献付出，无怨无悔！
　　一念嗔恨起，火烧功德林，人一旦有了嗔恨心，功德就没了，一旦抱怨别人，说别人的坏话，功德就没了，口业的罪过很大。
　　【7】福田：
　　没有福田，也很难成就事业。但是内心不是为了功德，而是为了众生，抬头三尺有神明，我们起心动念周围的一切众生都知道的，你内心的一个清净的善念，也可以感动周围的众生来帮助你，同样我们不要起任何一个恶念，哪怕是一个小小的。
　　没有慈悲心，求什么都白搭。真真正正的利益众生的心和行，功德是很不可思议的，菩萨绝对没有自私自利的心，追求功德的心。如果是为了功德，那功德就小了
　　在福田方面，我有四个好的建议：
　　1：放生：放生是功德很大的，内心是真的想要帮助众生，那样去放生，功德不可思议！
　　2：供僧：现在很多藏地的寺院都非常缺水缺粮食，藏地的修行人修行的环境很艰苦。同时，供僧的功德是非常大的。我曾经在我家里发生经济危机的时候供养我的上师100元，我的家庭马上就好转了。大家可以供养五明佛学院和亚青寺的僧众饮食衣服，相关的途径在地藏缘论坛中有，亚青寺现在在打井，解决喝水的问题，这可是殊胜的福田啊！
　　3：吃素：吃素也是在保护众生，一般吃素的人身体都比较轻，脾气柔和，吃肉的人办事不顺利，淫欲增强，护法善神很不喜欢。并且我们所吃的众生都会成为我们的冤亲债主，在吃素的同时也不能吃葱，蒜，韭菜，这是五辛，是污秽的东西，读经念咒期间更是不能吃的。
　　4：法布施，弘扬佛法，印经书，弘扬戒邪淫！
　　我举个例子：
　　1：见到小狗小猫，过去给它念一句观音心咒让它听到，来世它可以解脱三恶道，这也是法布施哦。要知道，旁生道听闻佛法有多么不容易。碰到流浪的受伤的众生，有能力就帮帮它，没有能力就回向它一部地藏经或者金刚经吧。
　　2：如果你有什么好的佛学资料或者好的戒淫方法，请发到论坛和大家共享，劝导你身边的人戒邪淫，或者给你身边的人看净空法师讲的《和谐拯救危机》的光盘，我这里很多不信佛的看了都信佛了！
　　最后，话说回来，不论你做什么功德，都不要执着不要贪着，这一点金刚经中讲了：菩萨所做功德不应贪着！真真正正的利益众生的心和行，功德是很不可思议的。
　　【8】信心：祈祷加持的感应最主要的就是来自己于信心，意思就是当你向三宝祈祷某个善愿的时候，在你的内心要深信无比的相信：此刻，我已经得到了三宝的加持我对佛菩萨不怀疑，我的任何就是修行，不要在修行的时候还在怀疑我能不能满愿啊如果你这样想，效果就差很多了。
　　不论你求什么愿望，不论是对婚姻还是家庭，只要你犯下下面这三个严重的错误，求什么都白搭，忏悔改正吧，这三件事，是非常消福报的
　　1：不孝父母，导致所有的事不顺。供养承事父母积累福报
　　2：邪淫，导致婚姻事业不顺利。我想这个问题大德们说了很多遍了。你想有好的婚姻事业吗，请猛烈忏悔邪淫，发愿努力改正
　　3：浪费粮食：现世损财运，消福报！来世堕恶道，没饭吃，高僧大德以及古圣先贤都说：浪费粮食遭天谴！
　　净空法师也讲过，现在全球10亿人吃不上饭，尤其是浪费大米更是罪上加罪。浪费粮食一般是现实损财运，来世没饭吃，没衣服穿。
　　倘若你的和你的亲人真的就吃剩的吃不了了，或者在饭店里你和你的亲人吃不了了，那就带回去给楼下的蚂蚁或者老鼠吃。或者合适的食物给小鸟吃，功德无量。
　　同时，你看到讨饭的人，给他买点东西吃，或者供养寺院或者藏地的寺院饮食衣物，这些都是可以忏悔浪费的罪业，减轻或消除果报。
　　【9】发愿：
　　发愿很重要。你要求伴侣，应该发善愿，佛菩萨也只会满足人的善愿，我要发愿求一个能帮助我修行的，和我一起做善事的伴侣，我得到这个伴侣后一定会精进修行，早日成就度化众生！
　　我真心希望：没出家的佛子都能早日组建佛化家庭，早日成就度化众生。
　　修行在于坚持，不是一朝一夕的，求愿也是同样，坚持下去，不要气馁。努力修行修善修德，天道酬勤！

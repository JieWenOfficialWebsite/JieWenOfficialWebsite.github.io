戒文~放生释疑—回答一个常见的放生疑问

一、放生的疑问
请问我们每次放生，常常要到鸟店买鸟，因为鸟店就只有固定那几家，买久了，鸟店老板也就知道我们会去买鸟来放生，便事前故意去屯积，去调货，准备好很多很多的鸟等我们去买。如此放生，早被人家算计中了，也同时助长了人家去抓鸟来给我们放，好像放生无形中更助长了别人的杀业，很多人都在骂我们笑我们愚痴，连我自己心中也有所疑惑，我实在不敢再去买鸟来放生了，是否能给我一些宝贵的意见？
二、既普遍又悲哀的现象
你问的这个问题，是一个非常普遍的现象，也是一个非常悲哀的现象。不仅是一般人对放生多所指责批评，连自己已发心放生，甚至放生有一段时间的居士们，也常因为自己的无明业障，因为自己缺乏智慧，自己对放生善举产生困惑，甚至对教导自己放生的善知识也产生怀疑，不自觉地被怨亲债主阻挠，被魔王邪众蛊惑，自己用自己的愚知愚见百般推理，再加上乱听别人用似是而非的学问逻辑或者引用一些有名有势的大人物或者专家、教授、学者的一些反对放生的言论，渐渐地退失了放生的初心，放弃了偿债赎罪的大好机会，这种人多不胜数。
三、守住当下第一念的慈悲
放生有没有学问？放生有很多学问，须要注意的地方很多，这是对刚接触放生的人讲的。刚接触放生的人，须要常常注意放生过程中物命的安危，放生地点的选择适不适合，放生后应多待一下，以免随后有恶人尾随捕猎，伤及物命…等等，这些都须要透过实际的行动参与慢慢学习，因为这是放生，是救命的行为，我们应当小心审慎！
但是放生究竟有没有学问？其实放生根本没有学问（这个根本没有学问就是最大最高最深的学问）。其实放生只是最最单纯的一念慈悲心而已！
放生就是我们见物命众生受擒受抓，失去自由，行将被砍被杀之际，一如见到自己父母手足妻女般，不忍见其苦难，遭受杀戮，发起慈悲心，予以救赎，还其自由，放其生命如此而已！
只是单单纯纯地，由内心里发起救赎生命的慈悲心，放生根本没有什么如法不如法，放生就是放生，放生就是观世音菩萨的大慈大悲，放生就是普贤菩萨的大行大愿，没有什么学问，只是一念清净的慈悲心而已，不须要用凡夫的愚见去自我思忖有没有如法，是不是人家故意抓来卖我，放了以后会不会活…，只要记得‘守住当下第一念的慈悲’，其他第二念，第三念，乃至无穷无尽的念头都是虚妄，都是自己的无明业障在作祟！
四、妄念阻断千千万万的生机
我们放鸟，到了鸟店，不忍鸟儿受擒被关，发慈悲心救他们，予以放生，一件单纯的功德当日便可以完成，无数的鸟儿便可以重拾生机，在大自然开创自己另一段的生命。
但如果我们自己愚痴想成‘如果我们不放，他们便不会去抓’，单单这样一个妄念的阻挠，当日那么多的鸟儿，就因为这一个念头，丧失了回复自由，重拾生命的机会，我们也自己断送了大好偿债赎命的契机；以后一传十，十传百，渐渐地也没有人愿意去买鸟来放生了！果真如此，所有受擒受抓的鸟儿当哭号悲泣，所有龙天护法当捶足钝胸，所有诸佛菩萨当叹息垂泪！而我们的所有怨亲债主当燃炮庆祝，魔王邪众当笑得合不拢口！愚痴的我们却自以为是，以为自己守法如法，无明障住了无数的生命与我们自己的前程，我们却浑然不知！
五、宛如大赦的放生行动
鸟店的老板如果料到了我们会去买鸟放生，因而事先屯积，到各地批发搜购，静静地等待放生人士前来采买，这样的话更值得大买特买，这有什么好烦恼犹豫的！只要能力许可，再多再多的鸟更应当买来放生。之前的放生一如救零零星星的死刑犯，鸟店老板为了赚钱，拼命去调各地被抓被关的鸟，集合在一起给我们放生，就如同总统大赦一样，全国的囚犯，都将因此重获自由！你仔细想想，因为我们放生，鸟店老板动用各种管道，为了赚钱想尽办法，把各地被抓被擒的鸟全部送来，等待我们解救放生。所有苦难的鸟儿当欣喜若狂，所有龙天护法当手舞足蹈，所有诸佛菩萨当满心欢喜，而我们所有的怨亲债主当颓头丧气，魔王邪众当气得横眉竖眼！如此天大的功德，如东我们却因‘怕别人算计了，预先准备给我们买’这种妄念，遂自我放弃，自我阻碍，却愚痴地自以为自己很如法，庆幸自己放生放得有学有问，有法有则，这就是最大最大的不幸！
六、鸟店一如等待大赦的囚犯集中营
我们稍微用脑筋去想一想，如果今天我们不再放生，不再放鸟了，那些捕鸟的猎人就从此不抓鸟了吗？卖鸟的鸟店就从此关门，生意就做不下去了吗？答案是否定的。鸟照样抓，捕网照样撒，鸟店照样开。漂亮名贵的鸟，被抓去买卖，关入牢笼，供人赏玩；外形枯瘦，价值不高的鸟可能被卖去炖补烧烤，或者一串串当成‘烤小鸟’沿街叫卖，更可能因为一文不名，落入网中，猎人只挑有价值的转卖，不值钱的便被随手脖子一拧，陈尸于荒郊野地之中。
如果真有专卖放生的鸟店，那也无妨，你只管去买，只管去放，放生就是放生，放生绝对绝对是一件好事。放生的鸟店，宛如等待大赦的囚犯集中营，鸟儿一则以喜，喜的是可能有机会值遇善人放生受救，鸟儿一则以惧，惧的是我们打消了放生念头，推三阻四，他们就一点存活的机会都没有了！
你当下退转了放生的念头，（不管是任何理由），当下那些鸟就失去了重获自由，重拾生命的机会，甚至连带地以后他们的同伴更将被无情地杀害，连一丝送到鸟店等著被放生的机会都没有！
七、别让知识学问框住了慈悲心
我们学习放生，所有的知识学问，经验方法就只是帮助我们放生，成全生命，长养慈悲而已，那些知识只是为了协助我们发挥慈悲心去解救众生的苦难，而不是拿那些知识来框住我们，限制我们的慈悲，阻碍我们去放生。当我们真正放生时，所有一切的知识学问都应舍掉，都应放下，只留那一念我们对待众生清清净净的慈悲，这才是放生真正的意义。
正如同我们被教导一定要在合法的医院，有合格的医生，有完善的设备才能进行手术，但是在战乱中，在蛮荒里，如果一定还要坚持万事俱全才打算进行手术，却无视于眼前奄奄一息，命在垂危的伤重生命，如此‘守法守纪’，如此‘如法而行’，岂不是颠倒至极！
原本放生一念清净的慈悲心，却因无知的凡夫妄加扭曲，才有如今那么多的怀疑，那么多的毁谤，甚至那么多的阻挠，那么多的轻蔑，这真是吾人当痛切反省的！
八、提倡放生的善知识难遇难逢
最令人感到悲哀的是，如果没有人教你放生，指导你放生，那你不懂放生，不会放生还情有可原！如果有位非常难得难逢的善知识，教你放生，指导你放生，而你还不肯放生，或者放生放了一段时间，对放生产生质疑退转，甚至对善知识产生毁谤怀疑，更有甚者，自以为是，不听从善知识的指导，以为自己是个专家，以盲引盲，乱用愚知愚见去渲染放生，误导放生，却不知自己与周遭同伴都已沦入魔道，落为魔子魔孙而不自知！
放生一事，看似非常简单，其间的道理非常深奥，非真正明眼具足智慧的人无法明白！非真正有修为具足定见的人无法提倡！试观自佛陀以来，历代提倡放生的诸祖大德，那大位不是行持操守，知识见地炫耀古今而受万人景仰，从随朝的智者大师，到宋朝的永明大师，到明朝的莲池大师，憨山大师到民国印光大师，虚云老和尚，那一位不是苦口婆心广劝世人放生，并且亲身实践力行放生！
九、井底之蛙愚知愚见
而今末法式微，众生业重福薄，善知识几不可见，而提倡放生的善知识更是少之又少，若有千载难逢的机会得遇一位提倡放生的善知识，我门一定要好好把握，竭心护持，奉事请益。
奈何吾辈凡夫却又瞎眼颠倒，听从邪师邪友，不愿意放生，批评放生者有之，自生我执我慢，对于放生，听不进善知识的指正，自生异见，自以为是者有之。殊不知我们只是业地凡夫，根机愚劣，善知识的修持见地，不是吾人所能明了于万一的，对于放生的真谛，也不是吾人所能了解于万一的。
如果你怀疑放生，怀疑善知识，那你也等于怀疑智者大师、永明大师、莲池大师、憨山大师、印光大师、虚云老师尚？甚至你更等于怀疑至高无上的佛陀了，因为放生正是佛陀亲口告诫我们长养慈悲，消罪消愆的最好方法。你以为自己是什么人，不知虚心请益，不知惭愧自省，还以为自己有独到的见解，而不听从善知识的指导，井底之蛙，小知小见，愚知愚见，莫此为甚，还不赶快忏悔认错，改邪归正，护法神是饶不了你的，因果报应也是逃不了的！
十、印祖警训
印光大师在《南浔极乐寺重修放生池疏》中早已明明白白地训勉我们：‘戒杀放生之事，浅而易见。戒杀放生之理，深而难明。若不明其理，纵能行其事，其心绝不能至诚恻怛。其福田利益，亦随其心量而致成微浅。倘遇不知者阻诽，遂可被彼所转，而一腔善心，随即消灭者有之。’
放生一事，事虽简而理甚深，到底不是我们鄙俗凡夫所能真正体会了解的。但只要我们坚持放生，信愿放生，跟随佛陀，跟随历代诸祖大德，跟随善知识的步伐，努力实践，终生不悔不倦，对于我们的生命，必能积聚无穷无尽的福善因缘，对于我们念佛往生的道业，更将有深远的影响！
但愿所有世人，广行放生；更愿所有念佛人，常行放生；因为放生功德，最为第一，舍此不行，是谓痴狂！

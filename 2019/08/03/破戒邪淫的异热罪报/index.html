戒文~破戒邪淫的异热罪报

Posted on 2019-08-03 
破戒邪淫的异热罪报<br/>　　<br/>节录自多洛尊者见闻记<br/>　　<br/>　　无悲心地狱的附近，是俱火浆地狱，此地有一座火浆形成的大山，周长一千逾缮那，山峰直冲天空，山上长着很多用燃烧的武器组成的树林。以牛头大狱卒为首的十万狱卒，将那些因命运而来到此地的暴风雪般的无数众生，一个个挂在武器树上。这些武器从众生的头部刺进去，从肛门穿出，众生因痛苦而大声惨叫。当大风将这些众生吹至山脚时，长着铁齿的许多罗剎女将他们紧紧抱住，咬吃他们的乳头，狱卒们用燃烧的铁绳将他(她)们捆在一起，使他们经受痛苦。<br/>　　<br/>　　在那里，我见到一个名叫阿旺格列的僧人和一个叫腊措的女人。僧人的生殖器已被狱卒们用铁钳拉出，女人的生殖器也被灌入铁水，都在承受无限的痛苦。<br/>　　<br/>　　我问牛头狱卒：「为什么这山上聚集着暴风雪般的无数众生？」<br/>　　<br/>　　狱卒说：「这是他们在人间时，破戒作乐的异熟。」<br/>　　<br/>　　我又问：「这些众生何时能从此地解脱？解脱之后又会生往何处？」<br/>　　<br/>　　狱卒说：「在业障没有消失之前，他们无法得到解脱，一旦解脱，他们也会生往寒地狱，将继续受苦……」<br/>

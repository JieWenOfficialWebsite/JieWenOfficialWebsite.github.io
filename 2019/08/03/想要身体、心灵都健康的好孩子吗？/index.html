戒文~想要身体、心灵都健康的好孩子吗？

我发此文的目的是：

愿天下新婚夫妇，能诞生出善男子善女人，感召好孩子来到世界上！对准备怀孕的年轻夫妇，真心奉劝的几句话。

自从结婚那日开始，如果您希望一生为孩子的事省心省力，建议您要记住：

在怀孕前要做好充分准备，不能让孩子是作为避孕失败的产物而出生。因此，要注意节育措施和节欲。

但是，一旦怀孕，就必须生下。那是生命，是与你们有缘的人；如果堕胎，就是杀人啊，后果不堪设想。

比如，到中年后，妇科疾病发病率，明显高于从没堕过胎的女人，乳腺癌、子宫肌瘤、宫颈出问题等等，很可怕！

迎接一个新生命的到来是一件多么重大的事情，因为他（她）将改变您的生活！决定您后半生的幸福！

为怀孕做好准备，是指要好好调养身体，注意饮食调养和做好感召有福报德善业众生作准备！

应做的准备如下：

1、食素

记住：并非吃鱼、吃虾、吃肉，才能培养出健康孩子，恰恰相反，那样吃生命，只会导致相反的结果！

建议吃如下食品：

首先，把买肉的钱用来买素食吧！

黄豆或其他豆类制品：豆腐、豆干、腐竹、豆浆、豆粥、豆菜随便吃吧！

萝卜、青菜，随便吃吧！

菜能生吃就生吃！

生的菜里含有大量鲜活的维生素和矿物质及微量元素，好好洗干净，沾酱（海天黄豆酱配上一点麻酱特别好吃），能生吃的就生吃，尽量别油炸别长时间煮，营养没啦！

主食要多吃，尽量多吃杂粮，可以吃杂粮豆粥或饭，红薯、土豆、干果、山药多吃。

还要多喝水。孕妇代谢快，等赶到渴了再喝已经晚了。

水果多吃！

但不能吃过热过凉的食物，子宫涨大了后就挨着胃呢，胎儿受不了。冷饮禁吃！

2、注意让眼根清静

这点，尤其重要。

其实，人的眼睛，是造成发生内心变化的一个窗户。

常言道：眼不见心为净。不要看任何凶残、暴戾、色情的场景。

尤其是不要看：黄色录像、色情画面、色情小说、娱乐绯闻等等，一切让你心动产生不良反应的视觉，坚决不看，不让内心受任何不净侵染。

能发誓坚持吗？如能，恭喜你，你为孩子做好了使其眼睛明亮，内心纯净智慧的基本资粮！

3、善护口业

让内心宽宏、柔软起来，多体谅一切人，你的老公、婆婆、同事、领导等等，不两舌、不恶口，不骗人。每说一句话都是在利益他人，与人玫瑰，手留芬芳。能做到吗？

如能，恭喜你，你的孩子一生都口出莲花，人见欢喜，受人赞叹！

4、善护意业

作为孕妇，平时该怎样思想？

那就在怀孕前就开始学佛吧，佛学，是人间最好的思想教育，揭示了世间一切福祸的原因，和趋福避祸的道理。

持诵经典，如果时间宽裕呢，就持诵《地藏经》，如果工作家务很忙呢，就建议持诵《观世音菩萨普门品》、《心经》。

如果您克服了懒散，挤时间定时定量坚持诵经，恭喜您，您若生了男孩，一定是利根聪明、福德智慧；如果是女孩，一定是宿值德本，端正有像，人见钦敬！

5、绝不杀生供养胎儿

天下的众生灵，无不避死求生，其实我们碗里的肉大家吃起来很欢喜，其实它是怎么搬上桌的你想过吗？很残忍！

您孩子的生命固然尊贵，但绝不应以牺牲其他生命作为代价！怨冤相报何时了！

如果您不仅吃素，还不杀任何生灵，包括苍蝇蚊子，还能在怀孕前后大量放生，恭喜您，您的孩子，将不爱生病，天生善良，富有爱心，人见人爱！一生都有好人缘！

但如果反其道行之，是什么恶果呢？如果您夫妇二人六根不净，放情恣意，或是随意堕胎、杀生食肉……..。我不想多说，您的果子自己慢慢品尝吧，人生长着呢！我也建议有儿有女的过来人回想一下，当逐渐感觉孩子是你的痛苦的时候，是否会想过自己再怀孕前后和育子过程中，自己是怎样做人的？但是不论何时，意识到是我们自己做错的时候，发忏悔心都不晚，尽快回来，开始接受佛学教育，那里告诉你一切痛苦的因缘和改善纠正的方法！

因果报应丝毫不爽！

怎样做最好呢？

总结以下几点，希望大家互相转告：

1、婚后前半年不要恣情纵欲，好好保养身体，接触佛教，为孩子做好准备。如果恣情纵欲不能控制，那么您就什么时候想要孩子，什么时候再开始用一定的时间做以上准备。

2、孕妇必须眼根清静，绝不许沾染任何不良视觉刺激，尤其以色情、血腥刺激切忌！！

3、尽量吃素，放心孩子发育不良，绝不是没吃肉的原因。优质蛋白不在肉里，在豆里！！！

4、少说话，如果不可避免，说出话来一定利益他人。

5、开始学佛，看佛教书，建议每日读诵《普门品》或《地藏菩萨本愿经》。

结论：

愿大家都生出福德深厚的好孩子，每对夫妇都做出努力，少一点放逸，多一点持戒，都会感召来品行端正、智慧圆满的健康宝宝来到人间，人间该多么美好！哪会有那么多的可怕事情发生！真正的和谐社会还会遥远吗?
